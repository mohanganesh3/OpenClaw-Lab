## Core Claim

This paper proposes an adaptive expert-weight-based dynamic routing scheme to effectively mitigate load imbalance in MoE models.

## Limitations

- Existing methods using auxiliary loss functions require manual hyperparameter tuning for different tasks.
- There is a trade-off where increasing activated experts worsens load imbalance, while fixing the count reduces model confidence on difficult tasks.
