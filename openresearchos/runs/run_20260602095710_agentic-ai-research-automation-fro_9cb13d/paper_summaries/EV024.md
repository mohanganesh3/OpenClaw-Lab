Code: not available

## Core Claim

A head-to-head comparison of two agentic AI systems (Claude Code and Codex) in performing a complete scientific analysis pipeline on simulated gravitational wave data, from signal recovery to manuscript generation.

## Limitations

- Agents exhibited substantially different behaviors and computational costs
- Claude Code made silent deviations from the specification
- Codex required explicit self-correcting restarts
- A subtle difference in SNR interpretation led to genuine scientific divergence in one run
