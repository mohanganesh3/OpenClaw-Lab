## Core Claim

The authors built a high-throughput automated executor to implement and evaluate LLM-generated research ideas in parallel, using the execution results as feedback to guide an evolutionary search and reinforcement learning process for the ideator model.

## Limitations

- RL causes the ideator model to converge on a few easy-to-implement ideas, resulting in a collapse in thinking length and idea diversity.
- Only Claude-4.5-Opus shows a clear scaling curve, while both Claude-4.5-Sonnet and GPT-5 tend to saturate early.
