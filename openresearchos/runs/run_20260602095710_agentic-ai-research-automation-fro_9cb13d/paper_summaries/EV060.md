# EV060: Physics-Informed Residuals for Adaptive Mesh Refinement in Finite-Difference PDE Solvers

Year: 2026
URL: https://arxiv.org/abs/2606.02475
Full Text: arxiv_html_openclaw
Datasets: One-dimensional viscous Burgers equation, 2D nonlinear Schrödinger equation, 3D incompressible Navier-Stokes system
Baselines: Uniform refinement, PINN-threshold refinement, PINN-Dörfler refinement, Gradient indicator, Random refinement
Best Result: PINN-threshold refinement achieves relative L² error 0.021067 with 60 degrees of freedom, compared to 0.022617 for uniform refinement with 192 degrees of freedom
Code: not available

## Core Claim

Uses Physics-Informed Neural Network residuals as cellwise indicators for adaptive mesh refinement in finite-difference PDE solvers

## Limitations

- Gradient indicator remains slightly more accurate
- PINN residuals don't consistently outperform gradient or uniform baselines in 2D/3D tests
