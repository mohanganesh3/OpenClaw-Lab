## Core Claim

Paper2Agent transforms static research papers into interactive AI agents by extracting the associated codebase, setting up a reproducible environment, wrapping core analytical features as MCP (Model Context Protocol) tools, and deploying a remote server that an AI agent can connect to for natural language interaction with the paper's methods.
