## Core Claim

POLARIS is a modular orchestration framework for back-office automation that treats the process as planning over typed agents, using a Chain-of-Action planner for type-checked DAGs, a rubric-based selector for policy-aware selection, and guarded execution with validators and compiled policy checks.
