## Abstract

Industrial surface processing lines equipped with inline reflectance spectrometers continuously acquire massive amounts of spectral data, but most deployed analysis methods still rely on hand-crafted spectral features and small, carefully calibrated datasets. As a result, their performance degrades under distribution shifts caused by material batch changes, surface roughness variation, and sensor drift. In this paper, we propose a physics-guided big-data learning framework for automatic industrial surface state recognition based on reflectance spectroscopy. The framework integrates four key components: (1) a spectral big-data management and quality-aware sampling module that organizes multiline, multidevice reflectance spectra and balances surface states and acquisition conditions; (2) a physics-guided preprocessing stage that performs baseline correction and smoothness-constrained decomposition of each spectrum, embedding reflectance priors into the learning objective; (3) a self-supervised spectral encoder built on a one-dimensional convolutional neural network (1D CNN)–Transformer backbone, which is pre-trained via masking-based reconstruction and contrastive learning on large-scale unlabeled spectra; and (4) an active-learning-driven incremental training scheme that selects informative spectra from newly collected data for annotation and continuously adapts the model to new surface states and operating conditions. Extensive experiments on a large-scale industrial reflectance spectral dataset demonstrate that the proposed framework consistently outperforms classical chemometric methods such as principal component analysis– partial least squares (PCA–PLS) regression and support vector machine (SVM) models and strong 1D deep baselines in both classification accuracy and regression accuracy for key surface parameters. Moreover, the active learning and incremental update strategy achieves comparable performance using only a fraction of the labeled data, significantly reducing the calibration burden in real-world deployments.
