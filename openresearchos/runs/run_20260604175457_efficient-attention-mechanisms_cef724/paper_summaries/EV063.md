# EV063: MLE-bench: Evaluating Machine Learning Agents on Machine Learning Engineering

Year: 2024
URL: https://arxiv.org/abs/2410.07095
Full Text: arxiv_abstract_openclaw
Datasets: Kaggle competitions
Baselines: Human baselines from Kaggle leaderboards
Best Result: OpenAI's o1-preview with AIDE scaffolding achieves at least the level of a Kaggle bronze medal in 16.9% of competitions
Code: yes

## Core Claim

MLE-bench uses Kaggle competitions to evaluate ML engineering skills of AI agents, with OpenAI's o1-preview and AIDE scaffolding as the best-performing setup.

## Limitations

- Resource scaling for AI agents
- Contamination from pre-training
