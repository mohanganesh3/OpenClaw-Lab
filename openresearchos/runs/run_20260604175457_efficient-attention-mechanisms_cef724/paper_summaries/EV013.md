## Core Claim

We propose SMNet, a method for multi-drone detection and classification by monitoring flight control signals, which is more robust than existing video-transmission signal monitoring methods.

## Limitations

- Existing RF-based methods that focus on video-transmission signals (VTS) can be evaded by disabling the VTS.
