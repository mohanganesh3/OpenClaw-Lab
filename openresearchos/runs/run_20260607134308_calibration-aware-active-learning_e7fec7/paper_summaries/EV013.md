# EV013: Probabilistic Embeddings for Frozen Vision-Language Models: Uncertainty Quantification with Gaussian Process Latent Variable Models

## Problem
Deterministic embeddings from standard Vision-Language Models struggle to capture uncertainties arising from ambiguities in visual and textual descriptions and multiple possible correspondences between images and texts.

## Core Claim
GroVE achieves state-of-the-art uncertainty calibration by learning probabilistic embeddings from frozen VLMs using Gaussian Process Latent Variable Models without requiring VLM retraining.

## Key Contributions
- Proposes GroVE as a post-hoc approach to obtain probabilistic embeddings from frozen VLMs
- Develops a unified latent space using GPLVM with single-modal reconstruction and cross-modal alignment objectives
- Demonstrates superior uncertainty calibration across cross-modal retrieval, VQA, and active learning tasks

## Method
GroVE builds on Gaussian Process Latent Variable Models to learn a shared low-dimensional latent space where image and text inputs are mapped to unified representations, optimized through single-modal embedding reconstruction and cross-modal alignment objectives.
- Components: Gaussian Process Latent Variable Model (GPLVM), Single-modal embedding reconstruction objective, Cross-modal alignment objective
- Key innovation: Using GPLVM to model uncertainty in the latent space while maintaining cross-modal alignment, allowing the model to capture uncertainties from both latent space uncertainty and inherent prediction noise.
- Assumptions: Frozen VLM provides meaningful initial embeddings; Low-dimensional latent space can capture essential structure; Gaussian Processes appropriately model uncertainty

## Experimental Setup
- Datasets: COCO, CUB
- Baselines: KL-Divergence, JS-Divergence, Wasserstein-2
- Metrics: KL-Divergence, JS-Divergence, Wasserstein-2

## Results
Headline: KL-Divergence achieves 0.79±0.02 for Image to Text on COCO

| Method | Dataset | Metric | Value |
|---|---|---|---|
| KL-Divergence | COCO | Image to Text | 0.79±0.02 |
| KL-Divergence | COCO | Text to Image | 0.65±0.02 |
| KL-Divergence | CUB | Image to Text | 0.49±0.05 |
| KL-Divergence | CUB | Text to Image | 0.44±0.05 |
| JS-Divergence | COCO | Image to Text | 0.70±0.04 |
| JS-Divergence | COCO | Text to Image | 0.59±0.03 |
| JS-Divergence | CUB | Image to Text | 0.48±0.02 |
| JS-Divergence | CUB | Text to Image | 0.44±0.05 |

- Ablations: Choice of probabilistic distance metric (KL-Divergence outperforms JS-Divergence and Wasserstein-2); Noise level impact on uncertainty prediction (GroVE's uncertainty rises with increased noise)
- Improvement over baseline: GroVE achieves superior cross-modal alignment using KL-Divergence compared to JS-Divergence and Wasserstein-2 metrics

## Critical Reading
- Stated limitations: Existing approaches require learning probabilistic embeddings during VLM training, which demands large datasets; The method is designed for vision-language tasks and may not directly apply to tabular data
- Unstated weaknesses: No evaluation on medical data or noisy label scenarios; No discussion of computational complexity for small datasets; No analysis of how the method handles label noise; No adaptation for tabular data format
- Threats to validity: The method is designed for vision-language tasks, not tabular data; No evaluation on the specific domain (medical); No consideration for the small data regime mentioned in the project; No analysis of how the method handles label noise
- Open questions: How would this method adapt to tabular data?; How does it perform with noisy labels?; What are the computational requirements for small datasets?; Can the GPLVM approach be effectively applied to tabular data?
