## Problem
Evaluating machine learning agents on machine learning engineering tasks through a structured benchmark framework.

## Core Claim
MLE-bench provides a comprehensive framework for evaluating ML agents on ML engineering tasks, demonstrated through a scalar coupling prediction competition.

## Key Contributions
- Introduction of MLE-bench as an evaluation framework for ML agents
- Log-MAE evaluation metric for scalar coupling prediction tasks
- Structured submission and evaluation procedures for ML engineering benchmarks

## Method
The paper presents MLE-bench which evaluates ML agents on predicting scalar coupling constants between atom pairs in molecules using the Log of Mean Absolute Error (Log-MAE) metric, calculated per coupling type and averaged across types.
- Components: Log-MAE evaluation metric with mathematical formulation, Molecule-based train/test split ensuring no molecule appears in both sets, Multiple data sources including XYZ structure files, dipole moments, magnetic shielding tensors, Mulliken charges, and scalar coupling contributions
- Key innovation: The Log-MAE metric that provides equal weight to improvements across different scalar coupling types, with a floor of '1e-9' for MAE and theoretical minimum score of approximately -20.7232 for perfect predictions.
- Assumptions: The molecule-based train/test split ensures proper generalization evaluation; The provided data sources are sufficient for accurate scalar coupling prediction; The Log-MAE metric appropriately captures model performance across different coupling types
