# EV006: Reflexion: Language Agents with Verbal Reinforcement Learning

Year: 2023 | Relation: method_source | Repro: medium | Source: pymupdf_full (19p, 35 tables)
URL: https://arxiv.org/abs/2303.11366 | Code: no

## Problem
Language agents struggle to efficiently learn from trial-and-error without requiring extensive training samples and expensive model fine-tuning.

## Core Claim
Reflexion achieves significant performance improvements in language agents by using verbal reinforcement learning through self-reflection and episodic memory rather than weight updates.

## Key Contributions
- A novel framework using linguistic feedback instead of traditional RL weight updates
- Implementation of verbal reflection with episodic memory for trial-and-error learning
- Demonstrated state-of-the-art results on multiple benchmarks (91% pass@1 on HumanEval)

## Method
Reflexion is an iterative framework where an Actor generates trajectories, an Evaluator scores them, and a Self-Reflection model creates verbal feedback stored in bounded episodic memory to improve subsequent trials.
- Components: Actor model that produces trajectories through environment interaction, Evaluator model that computes scalar reward scores, Self-Reflection model that generates verbal feedback summaries, Episodic memory buffer storing reflective experiences
- Key innovation: The core innovation is using verbal reinforcement through self-reflection and episodic memory to enable efficient trial-and-error learning without weight updates or gradient descent.
- Assumptions: LLMs can effectively generate and understand verbal feedback; Episodic memory of limited size (1-3 experiences) is sufficient for learning; Scalar or language feedback signals can be effectively processed

## Experimental Setup
- Datasets: HumanEval Python, HotPotQA (100 samples)
- Baselines: Baseline, CoT (GT), ReAct
- Metrics: Pass@1 accuracy
- Compute: Not specified

## Critical Reading
- Stated limitations: Traditional reinforcement learning methods require extensive training samples and expensive model fine-tuning; Language agents face challenges in quickly and efficiently learning from trial-and-error
- Unstated weaknesses: Performance on domains with highly specialized terminology (like medical data) is unclear; No clear mechanism for handling contradictory feedback signals; May struggle with tasks requiring precise numerical computations; Uncertain effectiveness with very small datasets; Potential sensitivity to the quality of feedback signals
- Threats to validity: May not generalize well to domains outside of those tested; Episodic memory buffer might introduce biases based on previous reflections; Quality of reflections might degrade over time or with repeated trials; Potential overfitting to specific types of feedback
- Open questions: How does Reflexion perform on small, specialized datasets like medical tabular data?; Can the reflection mechanism be adapted for calibration-aware learning?; How does Reflexion handle noisy labels in the feedback signals?; What is the optimal size and structure of the episodic memory buffer?; How does the verbal feedback mechanism interact with probabilistic calibration?

## Gaps This Enables
- A new approach to active learning that leverages self-reflection
- Potential for calibration-aware learning through reflective feedback
- Methods to handle noisy labels by critically evaluating feedback
- Techniques for small dataset learning through iterative reflection

## Relevance To Topic
Reflexion offers a novel approach to reinforcement learning through verbal feedback that could potentially be adapted for calibration-aware active learning under noisy labels in small medical tabular datasets.
