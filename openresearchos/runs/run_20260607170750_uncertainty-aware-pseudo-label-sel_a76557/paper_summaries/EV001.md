## Problem
Current computer-aided diagnosis systems use oversimplified one-hot labels that ignore diagnostic uncertainty, forcing models to make overconfident predictions especially with noisy or ambiguous medical images.

## Core Claim
Leveraging neural network training dynamics to generate calibrated uncertainty-aware pseudo-labels improves multi-view uncertainty estimation and robustness in medical image classification.

## Key Contributions
- Introducing a framework that uses neural network training dynamics (NNTD) to assess sample difficulty and generate uncertainty-aware pseudo-labels
- Developing calibrated pseudo-label methods (Pseudo-T and Pseudo-D) that improve model calibration through temperature scaling and Dirichlet calibration
- Demonstrating superior performance on echocardiography classification tasks with improved calibration, selective classification, and multi-view fusion

## Method
The method records model predictions across training epochs for each sample, aggregates these predictions into pseudo-labels representing model uncertainty, calibrates these pseudo-labels using validation data through temperature scaling or Dirichlet calibration, and trains models on these uncertainty-aware pseudo-labels to enhance robustness.
- Components: Neural network training dynamics (NNTD) aggregation using averaged logits across epochs, Temperature scaling (Pseudo-T) for basic calibration using validation cross-entropy minimization, Dirichlet calibration (Pseudo-D) with class-wise scaling matrix and bias using regularization
- Key innovation: Using the evolution of model predictions during training (rather than just final predictions) to quantify sample difficulty and generate uncertainty-aware pseudo-labels that reflect the ambiguity encountered during learning.
- Assumptions: The model's predictions during training reflect the inherent difficulty of each sample; Calibration on validation set generalizes to training set; Averaging logits across epochs provides a stable representation of model uncertainty

## Experimental Setup
- Datasets: aortic stenosis classification dataset (ultrasound videos)
- Baselines: Vanilla, Abstention [6], TMC [12], RT4U [9], Pseudo-T (ours), Pseudo-D (ours)
- Metrics: MAE, ACC, ECE, AURC
- Compute: single GPU (implied by 3D CNN training)
