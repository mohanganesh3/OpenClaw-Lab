# EV018: Uncertainty-Aware Extreme Point Tracing for Weakly Supervised Ultrasound Image Segmentation

## Problem
Fully supervised medical image segmentation requires extensive pixel-level annotations that are costly and time-consuming, making large-scale annotation impractical for clinical workflows.

## Core Claim
A weakly supervised segmentation framework using only four extreme points as annotation can achieve performance comparable to or surpassing fully supervised methods while significantly reducing annotation cost.

## Key Contributions
- A weakly supervised segmentation framework leveraging only four extreme points as annotation
- An enhanced Feature-Guided Extreme Point Masking (FGEPM) algorithm incorporating Monte Carlo dropout-based uncertainty estimation
- A dual-branch Uncertainty-aware Scale Consistency (USC) loss and box alignment loss for spatial consistency

## Method
The framework uses SAM2 with bounding box prompts derived from four extreme points to generate initial pseudo labels, which are progressively refined using an enhanced FGEPM algorithm with Monte Carlo dropout-based uncertainty estimation, while enforcing spatial consistency with USC loss and boundary alignment with box alignment loss.
- Components: Initial pseudo-label generation using SAM2 with bounding box prompts, Uncertainty-aware Scale Consistency (USC) loss, Box alignment loss, Monte Carlo dropout-based uncertainty estimation, Feature-Guided Extreme Point Masking (FGEPM) algorithm
- Key innovation: The construction of a unified gradient-uncertainty cost map using Monte Carlo dropout-based uncertainty estimation within the FGEPM algorithm, which allows contours to be traced more precisely from extreme points and ensures pseudo labels become progressively more accurate during training.
- Assumptions: Four extreme points (topmost, bottommost, leftmost, rightmost) can be reliably identified as the only annotation; SAM2 can effectively segment lesions when provided with bounding box prompts derived from extreme points; Monte Carlo dropout provides meaningful uncertainty estimates for pixel-wise refinement

## Experimental Setup
- Datasets: BUSI (Breast UltraSound Images) dataset, UNS (Ultrasound Nerve Segmentation) dataset with 2,323 image-mask pairs
- Baselines: BoxInst, WeakPolyp, BTBB, Deep Geodesics, PAplusNet, Fully supervision
- Metrics: Intersection over Union (IoU), Dice coefficient
- Compute: Not specified

## Critical Reading
- Stated limitations: Limited to ultrasound images, may not generalize to other imaging modalities; Requires accurate identification of four extreme points for each structure; Uncertainty estimation relies on Monte Carlo dropout which may have limitations in capturing all sources of uncertainty
- Unstated weaknesses: No explicit handling of label noise in the pseudo-label generation process; The method may struggle with complex anatomical structures where four extreme points are insufficient; Potential bias introduced by SAM2's initial pseudo-label generation; Limited evaluation on diverse datasets and imaging conditions
- Threats to validity: Results may not generalize to other medical imaging modalities; Performance may degrade with inaccurate extreme point annotations; Uncertainty estimates might not reflect true model uncertainty; Limited assessment of robustness to varying levels of annotation noise
- Open questions: How does the method perform with fewer or more extreme points?; What is the impact of extreme point localization errors on final segmentation quality?; How well does the uncertainty estimation correlate with actual segmentation errors?; Can the approach be effectively adapted for classification tasks?
