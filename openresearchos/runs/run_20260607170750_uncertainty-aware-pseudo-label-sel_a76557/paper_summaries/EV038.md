## Problem
The paper addresses the adaptability issue of semi-supervised learning techniques to Few-Shot Class-Incremental Learning (FSCIL) tasks, where unlabeled data is incorporated to compensate for limited labeled samples but causes instability in knowledge distillation.

## Core Claim
The Uncertainty-aware Distillation with Class-Equilibrium (UaD-CE) framework effectively resolves Semi-FSCIL challenges by combining class-balanced self-training and uncertainty-aware distillation.

## Key Contributions
- Proposed UaD-CE framework integrating Uncertainty-aware Distillation (UaD) and Class-Equilibrium (CE) modules
- Implemented class-balanced self-training to prevent easy-to-classify classes from dominating pseudo-label generation
- Developed uncertainty-guided knowledge refinement with adaptive distillation for stable knowledge transfer

## Method
The framework operates in incremental sessions: (1) CE module uses class-balanced self-training to incorporate unlabeled data while preventing class imbalance; (2) UaD module performs uncertainty-aware distillation between reference and target models using exemplars; (3) exemplars are updated by filtering samples from current datasets. - Components: Class-Equilibrium (CE) module: Class-balanced self-training with pseudo-label selection based on class difficulty, Uncertainty-aware Distillation (UaD) module: Knowledge distillation with uncertainty handling, Exemplar management: Dynamic updating of old class samples for knowledge transfer
- Key innovation: The class-balanced self-training mechanism that selects unlabeled samples based on inverse class difficulty (Equation 3), ensuring hard-to-classify classes receive more pseudo-label samples to prevent dominance by easy classes during semi-supervised learning.
- Assumptions: Unlabeled data contains useful information for learning novel classes; Model uncertainty can be estimated from prediction probabilities; Class difficulty correlates with prediction confidence; Knowledge distillation can effectively transfer knowledge between models
