## Problem
Limited availability of annotated medical imaging data and the ineffectiveness of transfer learning from natural image domains to medical tasks.

## Core Claim
A lightweight architecture (MAKNet) trained with semi-supervised pseudo-labels achieves superior performance with 60-70% fewer parameters compared to popular architectures.

## Key Contributions
- Proposed MAKNet with mixed asymmetric kernels (MAKConv) for efficient feature extraction
- Semi-supervised learning pipeline using teacher-student framework to generate pseudo-labels from large unlabeled medical dataset
- Demonstrated importance of domain-specific knowledge for effective transfer learning in medical imaging

## Method
MAKNet uses dense blocks with MAKConv layers, followed by standard conv, CBAM attention, and Mish activation. Semi-supervised training employs a teacher network (15 epochs) to generate pseudo-labels (>0.5 confidence) for unlabeled data, training a student network on concatenated labeled/unlabeled batches.
- Components: MAKConv: Mixed asymmetric kernels (spatial, single-dimension, depthwise), Dense Blocks: 3-6 connected MAKConv layers with standard conv + CBAM, Semi-supervised Pipeline: Teacher-student framework with pseudo-label generation, Preprocessing: HU normalization, background clipping, 3-slice input channels
- Key innovation: Mixed asymmetric kernels (MAKConv) enable comprehensive multi-dimensional feature extraction with significantly reduced parameters, while semi-supervised pseudo-labeling leverages large unlabeled medical datasets for domain-specific knowledge transfer.
- Assumptions: Unlabeled medical dataset contains sufficient diversity for meaningful pseudo-labels; Confidence threshold (>0.5) and ontology filtering produce reliable pseudo-labels; Domain-specific features transfer better than natural image features
