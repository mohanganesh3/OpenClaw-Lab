## Problem
Surgical phase recognition requires expensive full annotations where surgeons mark precise start/end times for each phase, creating a barrier for practical deployment.

## Core Claim
Timestamp supervision (single timestamp per phase) with uncertainty-aware temporal diffusion achieves competitive performance to full supervision while reducing annotation cost by 74%.

## Key Contributions
- Introduces timestamp supervision paradigm for surgical phase recognition
- Proposes Uncertainty-Aware Temporal Diffusion (UATD) method to generate pseudo labels from timestamps
- Demonstrates loop training strategy to handle long videos and class imbalance

## Method
Uses Monte Carlo Dropout for uncertainty estimation, diffuses timestamp labels to adjacent low-uncertainty frames, and employs loop training to optimize spatial and temporal extractors separately using only labeled frames.
- Components: Uncertainty estimation via Monte Carlo Dropout, Temporal diffusion algorithm with uncertainty thresholding, Loop training decoupling spatial/temporal optimization
- Key innovation: Temporal diffusion stops at high-uncertainty frames (boundaries), ensuring generated pseudo labels are high-confidence and phase-consistent, enabling effective learning from minimal supervision.
- Assumptions: Surgical phases are long events with consecutive frames; Uncertainty estimation via Monte Carlo Dropout reliably identifies high-confidence frames; Surgeons tend to annotate timestamps near phase midpoints
