## Problem
Semi-supervised learning for medical image classification faces challenges with class imbalance due to scarcity of disease-positive individuals, which existing SSL methods don't adequately address.

## Core Claim
The Adaptive Blended Consistency Loss (ABCL) effectively counteracts data skew in imbalanced medical image classification by adaptively blending target class distributions based on class frequencies.

## Key Contributions
- Proposed Adaptive Blended Consistency Loss (ABCL) as a drop-in replacement for consistency loss in perturbation-based SSL methods
- Demonstrated improvements to unweighted average recall on two imbalanced medical image classification datasets

## Method
ABCL creates a blended target distribution that combines OSP and ASP predictions with weighting determined by class frequencies, pulling both original and augmented predictions toward this target distribution using Kullback-Leibler divergence loss.
- Components: ABCL loss function: L_ABCL(z, ŷ) = L_con(z_blended, z) + L_con(z_blended, ŷ), Blended target distribution: z_blended = (1-k) * z + k * ŷ, Weighting calculation: k = max(0, min(γ * (N_original - N_augmented) + 0.5, 1))
- Key innovation: Adaptive weighting of target distributions based on class frequencies to counteract natural bias toward majority classes, with k < 0.5 when OSP is minority class and k > 0.5 when ASP is minority class.
- Assumptions: The model's predictions for original and augmented samples can be meaningfully combined; Class frequency information can effectively guide the blending of target distributions

## Experimental Setup
- Datasets: HAM10000 (skin cancer dataset), REFUGE dataset
- Baselines: UDA baseline, Sampling (SMOTE and random undersampling), Weighted Cross Entropy Loss (WeightedCE), Focal Loss, Consistency Loss Suppression (SCL), UDA-WeightedCE-SCL
- Metrics: UAR (Unweighted Average Recall), G-mean, Average AUC, Individual class recall, ROC and AUC for each class
- Compute: Not specified

## Results
Headline: UDA-WeightedCE-ABCL achieved UAR of 0.68, G-mean of 0.66, and average AUC of 0.96 on HAM10000 dataset

| Method | Dataset | Metric | Value |
|---|---|---|---|
| UDA-ABCL(ours) | HAM10000 | UAR | 0.67 |
| UDA-ABCL(ours) | HAM10000 | G-mean | 0.62 |
| UDA-ABCL(ours) | HAM10000 | Average AUC | 0.95 |
| UDA-WeightedCE-ABCL(ours) | HAM10000 | UAR | 0.68 |
| UDA-WeightedCE-ABCL(ours) | HAM10000 | G-mean | 0.66 |
| UDA-WeightedCE-ABCL(ours) | HAM10000 | Average AUC | 0.96 |
| UDA baseline | HAM10000 | UAR | 0.59 |
| UDA baseline | HAM10000 | G-mean | 0.56 |

- Ablations: Selective versus always-on target blending; Effect of γ parameter variation; Impact of combining WeightedCE with ABCL
- Improvement over baseline: ABCL improved UAR from 0.59 to 0.67 (+0.08), G-mean from 0.56 to 0.62 (+0.06), and average AUC from 0.91 to 0.95 (+0.04) over UDA baseline
