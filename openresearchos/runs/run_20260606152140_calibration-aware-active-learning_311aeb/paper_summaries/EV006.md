## Core Claim

Reflexion is formalized as an iterative optimization process where an Actor produces a trajectory τ₀ by interacting with the environment, an Evaluator computes a score rt = Me(τ₀), and a Self-Reflection model generates a verbal summary sr₀ which is stored in memory for improvement across trials.

## Limitations

- Memory must be bounded by maximum number of stored experiences to adhere to max context LLM limitations
