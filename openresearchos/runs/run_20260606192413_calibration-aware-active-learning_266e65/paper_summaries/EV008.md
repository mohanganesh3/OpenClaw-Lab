## Problem
Creating embodied agents that can continuously explore, acquire diverse skills, and make novel discoveries without human intervention in open-ended environments.

## Core Claim
VOYAGER is the first LLM-powered embodied lifelong learning agent that achieves open-ended exploration and skill acquisition in Minecraft through its three key components, outperforming prior methods.

## Key Contributions
- An automatic curriculum that maximizes exploration
- An ever-growing skill library of executable code for storing and retrieving complex behaviors
- A new iterative prompting mechanism that incorporates environment feedback, execution errors, and self-verification for program improvement

## Method
VOYAGER interacts with GPT-4 via blackbox queries to generate executable code for actions in Minecraft, using Mineflayer as the low-level controller. The agent's three components work together to drive exploration, store skills, and improve programs iteratively.
- Components: Automatic curriculum for open-ended exploration, Skill library for increasingly complex behaviors, Iterative prompting mechanism that uses code as action space
- Key innovation: The integration of environment feedback, execution errors, and self-verification into an iterative prompting mechanism, with self-verification being the most critical component for deciding when to move on to new tasks or reattempt unsuccessful ones.
- Assumptions: GPT-4 provides sufficient code generation quality for the task; Minecraft's environment can be effectively represented through text-based feedback; The skill library approach can effectively store and retrieve complex behaviors; The automatic curriculum can effectively drive exploration
