## Problem
Missing instances in Earth Observation time series data due to satellite failure or cloud occlusion introduce input-level uncertainty that degrades regression performance while being commonly overlooked in existing approaches.

## Core Claim
Monte Carlo Concrete Temporal Dropout (MC-ConcTD) improves predictive performance and uncertainty calibration in EO time series regression by learning optimal dropout distributions rather than using fixed ratios.

## Key Contributions
- Introduction of Monte Carlo Temporal Dropout (MC-TD) for input-level uncertainty quantification using predefined dropout ratios
- Development of Monte Carlo Concrete Temporal Dropout (MC-ConcTD) that learns optimal dropout distributions via continuous relaxation

## Method
MC-TD applies random temporal dropout masks drawn from Bernoulli distribution during inference with fixed ratio α. MC-ConcTD extends this using Concrete distribution to learn α as a parameter via continuous relaxation, both using Monte Carlo sampling (L=20) for uncertainty quantification.
- Components: Temporal Dropout (TD) with binary masks from Bernoulli distribution, Concrete Temporal Dropout (ConcTD) with learnable parameters via continuous relaxation, Monte Carlo sampling for uncertainty estimation
- Key innovation: Using learnable dropout ratios through Concrete distribution to automatically optimize uncertainty quantification, eliminating costly manual ratio selection while improving model calibration.
- Assumptions: Input uncertainty propagates through the model to result in epistemic uncertainty; Dropout approximates Bayesian inference in neural networks; The dropout ratio can be optimized via gradient descent

## Results
Headline: MC-ConcTD achieves best R² of 0.79 on SwissYield dataset

| Method | Dataset | Metric | Value |
|---|---|---|---|
| MC-Dropout | SwissYield | R² | 0.78 |
| MC-ConcTD | SwissYield | R² | 0.79 |
| MC-ConcTD | PM2.5 | R² | 0.77 |
| MC-ConcTD | LFMC | R² | 0.72 |
| MC-ConcTD | SwissYield | RMSE | 0.93 |
| MC-ConcTD | PM2.5 | RMSE | 31.77 |
| MC-ConcTD | LFMC | RMSE | 20.80 |
| MC-ConcTD | SwissYield | PU | 1.23 |

- Ablations: Comparison of TD variants (MC-TD vs MC-ConcTD) and established UQ methods (MC-Dropout, VI)
- Improvement over baseline: MC-ConcTD improves R² by 0.01 on SwissYield (0.78→0.79) and 0.03 on PM2.5 (0.74→0.77) vs MC-Dropout
