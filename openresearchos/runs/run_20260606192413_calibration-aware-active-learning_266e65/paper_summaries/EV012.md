## Problem
Reward models in RLHF overlook epistemic uncertainty from limited human feedback, leading to reward hacking and alignment issues.

## Core Claim
RewardUQ provides a unified framework that systematically evaluates uncertainty quantification methods, revealing model size and initialization as the most impactful factors.

## Key Contributions
- Introduces RewardUQ framework for systematic UQ evaluation
- Compares four UQ architectures (ENS-MLP, ENS-LoRA, MCD-DPO, BAY-LIN) on accuracy/calibration metrics
- Proposes new ranking strategy combining accuracy and calibration
- Releases open-source Python package for UQ evaluation

## Method
Trains four UQ architectures using embeddings from pretrained LLMs, modifies standard losses for each method, and evaluates on standard metrics with a unified procedure.
- Components: MLP head ensembles with regularization terms, LoRA adapter ensembles for parameter efficiency, DPO-based Monte Carlo dropout for implicit reward uncertainty, Bayesian linear heads with Laplace approximation
- Key innovation: Unified evaluation framework enabling systematic comparison of UQ methods across accuracy, calibration, and ranking dimensions.
- Assumptions: Epistemic uncertainty can be quantified via model ensembles/dropout/Bayesian methods; Embeddings from pretrained LLMs capture relevant features; Standard accuracy/calibration metrics adequately evaluate UQ performance; Model size/initialization dominate over architectural differences
