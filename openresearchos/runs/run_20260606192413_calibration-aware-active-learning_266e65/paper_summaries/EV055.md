## Problem
Standard benchmarks cannot measure how close neural networks are to optimal performance because they lack access to the true posterior p(y|x), making it impossible to distinguish irreducible aleatoric uncertainty from reducible epistemic error.

## Core Claim
Access to exact posteriors via class-conditional normalizing flows reveals hidden learning dynamics, architectural differences, and enables precise quantification of model limitations and data efficiency.

## Key Contributions
- Demonstrates that epistemic error follows power-law scaling even when total loss plateaus, revealing ongoing learning beyond standard metrics.
- Shows architectures differ markedly in approaching the aleatoric floor, with ResNets exhibiting clean power-law scaling while ViTs stall in low-data regimes.
- Proves oracle posteriors contain learnable structure beyond class labels, enabling near-perfect calibration when used as soft labels.
- Establishes that distribution shift type matters more than magnitude, as class imbalance causes minimal accuracy loss at KL divergences where input noise causes catastrophic degradation.
- Shows exact epistemic uncertainty distinguishes informative samples from ambiguous ones, improving active learning sample efficiency.

## Method
Uses class-conditional normalizing flows as oracles to compute exact posteriors p(y|x) for realistic images, enabling decomposition of prediction error into aleatoric and epistemic components and exact KL divergence computation for distribution shift analysis.
- Components: Class-conditional normalizing flow oracles that make exact posteriors tractable, Decomposition of prediction error into irreducible aleatoric uncertainty and reducible epistemic error, Power-law fitting to quantify epistemic error scaling with dataset size, Exact KL divergence computation for controlled distribution shift perturbations
- Key innovation: Exact posteriors enable precise decomposition of neural network error into aleatoric and epistemic components, revealing hidden learning dynamics and architectural differences invisible to standard metrics.
- Assumptions: Class-conditional normalizing flows can perfectly model true posteriors; Datasets (AFHQ, ImageNet) are representative of real-world image distributions; Power-law scaling holds across architectures and datasets

## Experimental Setup
- Datasets: AFHQ, ImageNet
- Baselines: ResNets, Vision Transformers (ViTs)
- Metrics: KL divergence (epistemic error), accuracy, robustness to distribution shift
- Compute: 8x NVIDIA V100 GPUs

## Results
Headline: Epistemic error follows a power law decay (KL ∝ N^{-α}) across architectures and scales to ImageNet-1000 classes

- Ablations: —
- Improvement over baseline: ResNets reduce epistemic error by 6× over the data range where ViTs improve by only 1.2×
