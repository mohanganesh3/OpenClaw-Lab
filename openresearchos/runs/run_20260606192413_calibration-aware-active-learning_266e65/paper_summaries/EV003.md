## Problem
Scientific discovery is a lengthy and costly process that demands substantial time and resources from initial conception to final results.

## Core Claim
Agent Laboratory, an autonomous LLM-based framework, can complete the entire research process from literature review to report writing, producing comprehensive research outputs while enabling human feedback at each stage.

## Key Contributions
- Introduction of Agent Laboratory, an autonomous LLM-based framework for completing the entire research process
- Demonstration that Agent Laboratory can significantly reduce research expenses (84% decrease compared to previous autonomous research methods)
- Finding that Agent Laboratory driven by o1-preview generates the best research outcomes
- Finding that the generated machine learning code achieves state-of-the-art performance compared to existing methods

## Method
The method involves an autonomous LLM-based framework that accepts a human-provided research idea and progresses through three stages: literature review, experimentation, and report writing, producing comprehensive research outputs including a code repository and a research report, while enabling users to provide feedback and guidance at each stage.
- Components: Literature review stage, Experimentation stage, Report writing stage, Human feedback mechanism at each stage
- Key innovation: The autonomous LLM-based framework that can complete the entire research process from start to finish, with the ability to produce comprehensive research outputs and enable human feedback at each stage.
- Assumptions: That LLM agents can effectively perform literature review, experimentation, and report writing; That human feedback at each stage can improve the overall quality of research; That the framework can produce comprehensive research outputs including code repositories and research reports
