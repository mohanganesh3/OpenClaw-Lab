## Problem
Open-set active learning requires effective handling of unknown classes and stable high-precision queries, especially in early training stages where existing methods suffer from instability and suboptimal performance.

## Core Claim
The proposed method achieves superior query precision and test accuracy by dynamically estimating unknown classes, regulating purity via GMM, and using information-based selection, outperforming state-of-the-art baselines across benchmarks.

## Key Contributions
- A purity-based candidate selection framework using auxiliary head outputs and GMM-modeled purity regimes (high/known, low/unknown, intermediate/ambiguous).
- A flexible sampling strategy that adapts to unknown class estimates and maintains high query precision from early rounds without additional hyperparameters.

## Method
Trains a dual-head classifier (LCE for known classes, LEDL for unknown), computes purity margins using auxiliary head logits, fits a 3-component GMM to logit margins, refines candidates via calibrated precision targets, and selects queries using JS divergence scores.
- Components: Purity margin computation: Spurity(x) = maxc∈Ck oc (known) or maxc∈Ck oc − maxc∈Cˆu oc (unknown)., GMM fitting on logit margins to model purity regimes and likelihood-based sorting., Precision-based refinement using calibrated target precision ˆp∗t., Information-based selection via JS divergence: Sinfo(x) = JS(p∥u) · JS(p∥pmax).
- Key innovation: Dynamic purity regulation through GMM-modeled regimes and adaptive unknown estimation enables stable high-precision queries without manual hyperparameter tuning.
- Assumptions: Access to labeled pool DL and unlabeled pool DU.; Upper bound ˆumax for unknown class count estimation.; Availability of Algorithm 1 for unknown estimation.

## Experimental Setup
- Datasets: CIFAR-10, CIFAR-100, Tiny-ImageNet, ImageNet-1K
- Baselines: Random, Margin, Core-Set, BADGE, VAAL, Open-Set BALD, OSAL
- Metrics: accuracy, ECE, AUROC
- Compute: single GPU
