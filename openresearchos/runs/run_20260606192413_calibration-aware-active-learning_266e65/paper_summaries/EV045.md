## Problem
Computer-aided diagnosis systems are trained on overly simplistic one-hot labels that ignore diagnostic uncertainty, forcing models to make overconfident predictions especially with noisy or ambiguous medical images.

## Core Claim
By leveraging neural network training dynamics to generate uncertainty-aware pseudo-labels, we can enhance uncertainty estimation and robustness in medical image classification systems.

## Key Contributions
- A novel framework that brings uncertainty back into the label space using neural network training dynamics
- Architecture-agnostic pseudo-label generation method applicable to any supervised learning pipeline
- Calibrated pseudo-label approach using Dirichlet Calibration for class-specific scaling

## Method
The method records the history of predicted logits during training, averages them over epochs to create pseudo-labels, and then calibrates these labels using either temperature scaling or Dirichlet Calibration to ensure proper confidence calibration.
- Components: Neural Network Training Dynamics (NNTD) for pseudo-label generation: y'_i = (1/T) * Σ σ(f_t(x_i)), Temperature scaling (Pseudo-T) for basic calibration: y'_i = σ(γ* v_i), Dirichlet Calibration (Pseudo-D) for class-specific calibration: y'_i = σ(A* v_i + b*)
- Key innovation: The core innovation is using the model's own training dynamics - specifically how predictions evolve over epochs - to quantify sample difficulty and generate uncertainty-aware pseudo-labels, which captures inherent ambiguity in the data that traditional one-hot labels miss.
- Assumptions: The model's evolving predictions over epochs provide insight into sample difficulty; The validation set is representative for calibration purposes; Training dynamics capture the inherent difficulty of samples including task complexity, data issues, and label noise
