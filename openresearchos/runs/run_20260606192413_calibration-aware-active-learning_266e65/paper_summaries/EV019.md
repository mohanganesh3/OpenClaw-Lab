## Problem
Deterministic embeddings from standard Vision-Language Models (VLMs) struggle to capture uncertainties arising from ambiguities in visual/textual descriptions and multiple possible correspondences between images and texts.

## Core Claim
GroVE achieves state-of-the-art uncertainty calibration by generating probabilistic embeddings from frozen VLMs through a post-hoc Gaussian Process Latent Variable Model approach.

## Key Contributions
- GroVE framework for post-hoc probabilistic embeddings from frozen VLMs
- GPLVM-based shared latent space learning with single-modal reconstruction and cross-modal alignment
- Superior uncertainty calibration across cross-modal retrieval, VQA, and active learning tasks

## Method
GroVE builds a Gaussian Process Latent Variable Model (GPLVM) to learn a shared low-dimensional latent space where image and text inputs map to unified representations, optimized through single-modal embedding reconstruction and cross-modal alignment objectives.
- Components: Gaussian Process Latent Variable Model (GPLVM) for latent space learning, Single-modal embedding reconstruction objectives, Cross-modal alignment objectives
- Key innovation: The post-hoc application of GPLVM to frozen VLMs to generate uncertainty-aware probabilistic embeddings without requiring retraining or large datasets, enabling uncertainty quantification for powerful pre-trained models like CLIP.
- Assumptions: Frozen VLMs provide sufficient initial representations for latent space learning; Shared low-dimensional latent space can capture cross-modal relationships; Uncertainty can be effectively modeled through Gaussian Processes
