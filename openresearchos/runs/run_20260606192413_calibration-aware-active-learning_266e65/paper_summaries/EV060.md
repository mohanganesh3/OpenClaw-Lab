## Problem
The paper addresses the challenge of human-robot interaction where robots must anticipate human intent in ambiguous scenarios, particularly focusing on the problem that deep learning models for intent prediction are prone to being confidently incorrect.

## Core Claim
The paper presents Risk-Calibrated Interactive Planning (RCIP), a framework that statistically calibrates risk for uncertain action selection in human-robot cooperation, allowing robots to meet user-specified risk levels with specified probability over new scenarios.

## Key Contributions
- Development of Risk-Calibrated Interactive Planning (RCIP), a framework for measuring and calibrating risk in uncertain action selection for HRI
- Statistical risk calibration method that allows robots to meet user-specified risk levels with specified probability over new scenarios
- A practical approach that enables flexible autonomy while controlling multiple risks simultaneously (action miscoverage and human help rate)

## Method
The RCIP framework uses a calibration dataset to select parameter pairs (prediction thresholds λ and model parameters θ) that control user-specified risk levels. It employs a multiple-hypothesis testing approach using fixed-sequence testing to identify parameter sets that control the family-wise error rate across different risk metrics. The method generates prediction sets of actions and triggers human help when the set contains multiple actions.
- Components: Prediction Model: Uses models like Vision Language Models (VLMs) to predict intent probabilities, Intent-Conditioned Planner: Generates actions conditioned on predicted intents, Risk Calibration Module: Uses statistical methods to select parameters that control risk, Help Trigger Mechanism: Requests human assistance when prediction sets contain multiple actions
- Key innovation: The key innovation is the use of multiple-hypothesis testing with fixed-sequence testing to simultaneously control different risk metrics (action miscoverage and human help rate) by selecting appropriate prediction thresholds and model parameters, rather than just focusing on prediction accuracy.
- Assumptions: The robot has access to a calibration dataset drawn from the same distribution as future scenarios; The robot has ground-truth knowledge of the true intent z when available; The risk signal can be properly defined and measured; The prediction model produces softmax scores for each intent-conditioned action

## Experimental Setup
- Datasets: YCB dataset, Habitat simulator, Azure Kinect RGB-D camera
- Baselines: KnowNo [36], Simple Set, Entropy Set, No Help
- Metrics: Plan Success Rate, Execution Success Rate, Plan Help Rate, Step Help Rate
- Compute: Habitat simulator, Franka Emika Panda arm, Azure Kinect RGB-D camera

## Results
Headline: RCIP achieves 100% plan success and 100% step success with 0% help in Social Navigation (Model 3)
