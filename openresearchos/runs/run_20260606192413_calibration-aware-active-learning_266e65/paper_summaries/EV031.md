## Problem
Accurate EV power estimation underpins range prediction and energy management, yet practitioners need both point accuracy and trustworthy uncertainty quantification.

## Core Claim
An anchored-ensemble LSTM with Student-t likelihood jointly captures epistemic and aleatoric uncertainty, delivering accurate, well-calibrated predictions with efficient real-time inference.

## Key Contributions
- Anchored ensembles impose Gaussian weight priors via MAP training to generate posterior-like diversity without test-time sampling
- Student-t likelihood provides heavy-tailed robustness and closed-form prediction intervals for aleatoric uncertainty
- Achieves strong accuracy (RMSE 3.36±1.10, MAE 2.21±0.89, R²=0.93±0.02) and well-calibrated uncertainty bands with near-nominal coverage

## Method
Trains multiple LSTM models with gate-wise Gaussian priors anchored to learned weights, using Student-t negative log-likelihood for aleatoric uncertainty. Predictive intervals combine ensemble spread (epistemic) and t-distribution scale (aleatoric).
- Components: Anchored ensemble LSTM with gate-wise Gaussian priors (MAP objective), Student-t likelihood for heteroscedastic uncertainty estimation, Closed-form predictive intervals using ensemble averaging and t-distribution quantiles
- Key innovation: Anchoring imposes Gaussian weight priors during MAP training, generating posterior-like diversity without test-time sampling, enabling efficient uncertainty quantification.
- Assumptions: Input features are normalized to [0,1] range; Residuals follow Student-t distribution with ν>2 for finite variance; Gate-wise factorization of prior and likelihood enables independent quadratic penalties
