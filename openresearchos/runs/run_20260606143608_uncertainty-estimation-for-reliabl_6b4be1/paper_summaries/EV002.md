## Core Claim

The AI Scientist-v2 uses a progressive agentic tree-search methodology managed by a dedicated experiment manager agent to autonomously generate scientific manuscripts, eliminating reliance on human-authored code templates and generalizing across diverse machine learning domains, with an enhanced AI reviewer component that integrates a Vision-Language Model feedback loop for iterative refinement of figures.
