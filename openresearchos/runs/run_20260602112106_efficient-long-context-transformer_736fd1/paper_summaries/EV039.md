## Core Claim

This paper presents a low-dose CT denoising network based on adaptive global context attention to overcome the limitations of traditional convolutional methods and standard global context models.

## Limitations

- Traditional deep learning methods have small receptive fields and cannot capture long-range correlations.
- Standard Global Context (GC) attention models use a single global context feature, which is insufficient for describing non-homogeneous structures.
