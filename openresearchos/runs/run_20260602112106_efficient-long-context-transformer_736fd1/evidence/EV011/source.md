# EV011: Low-Latency Multimodal Semantic Ranking for Real-Time Advertising Recommendation

## Abstract

Large-scale online advertising recommendation needs both high accuracy and low latency. Traditional methods often fail to capture semantic and multimodal user–ad relationships under time limits. This paper presents LMSR (Low-Latency Multimodal Semantic Ranking), a two-stage framework that uses large language model (LLM) semantics, multimodal content understanding, and multi-scale behavioral modeling. The framework applies multi-channel recall with collaborative filtering, semantic retrieval, and sequence pattern analysis. It then performs hierarchical Transformer-based ranking with adaptive multimodal fusion. LLM representations are distilled into a small encoder to improve speed while keeping semantic quality. LMSR also uses multi-task learning with adaptive weighting to balance accuracy, diversity, and speed. The combination of LLM semantics, hierarchical sequence modeling, and adaptive multimodal fusion helps LMSR achieve better performance and real-time efficiency for advertising recommendation.
