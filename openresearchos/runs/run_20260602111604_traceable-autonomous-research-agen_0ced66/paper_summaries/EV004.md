# EV004: MLE-bench: Evaluating Machine Learning Agents on Machine Learning Engineering

Year: 2024
URL: https://arxiv.org/abs/2410.07095
Full Text: arxiv_abstract_openclaw
Datasets: Kaggle competitions
Baselines: Human baselines from Kaggle leaderboards
Best Result: OpenAI's o1-preview with AIDE scaffolding achieved at least the level of a Kaggle bronze medal in 16.9% of competitions.
Code: yes

## Core Claim

Created a benchmark using Kaggle competitions to evaluate ML agents on real-world ML engineering tasks, using human baselines from Kaggle leaderboards and open-source agent scaffolds.
