## Core Claim

The paper converts the preference poisoning problem into a structured binary sparse approximation problem and develops two attack methods: Binary-Aware Lattice Attack (BAL-A) using lattice reduction algorithms and Binary Matching Pursuit Attack (BMP-A) adapting binary matching pursuit to non-normalized gradient dictionaries.
