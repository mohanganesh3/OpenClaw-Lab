## Core Claim

The proposed OrthogonalMambaMIL method improves whole slide image classification by addressing the limitation of existing models that fail to learn the 2D spatial layout of patches.

## Limitations

- Existing attention- and Mamba-based models arrange patch embeddings into a 1D sequence, losing the original 2D spatial layout.
