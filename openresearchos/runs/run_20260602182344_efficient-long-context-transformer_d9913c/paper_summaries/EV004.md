## Core Claim

A low-dose CT denoising network based on adaptive global context attention is proposed to overcome the limitations of traditional convolutional methods.

## Limitations

- Small receptive fields in traditional methods
- Inability of traditional methods to capture long-range correlations
- Single global context feature in GC block cannot describe non-homogeneous structures
