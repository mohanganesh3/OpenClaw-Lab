## Core Claim

RoPeSLR is a 3D RoPE-guided Sparse-LowRank attention framework that addresses the RoPE Dilemma by using a head-wise low-rank parameterization with learnable 3D Absolute Positional Embedding injection to preserve the orthogonal relative-position structure of 3D Rotary Position Embeddings.
