## Abstract

This paper introduces the ConvolutionAugmented Agent Attention (CAAA) framework for solving the Colored Traveling Salesman Problem (CTSP). It addresses the reliance of traditional heuristics on expert knowledge and the challenge neural solvers face in balancing efficiency with solution quality. By proposing a linear-complexity Agent Attention architecture, global interactions are mediated through learnable agent tokens, reducing computational complexity from $O\left(N^{2} d\right)$ to $O(N n d)$. Additionally, the Convolutional Augmentation module enhances solving capability via local geometric feature extraction and crosslayer fusion. Experimental results show that CAAA matches the performance of exact solvers on small-scale CTSP instances and outperforms neural baselines in large-scale scenarios, while significantly reducing computation time compared to heuristic methods. With its reduced computational complexity and faster inference speed, CAAA offers an efficient and generalizable solution for heterogeneous path planning, marking a significant advancement in tackling complex routing problems.
