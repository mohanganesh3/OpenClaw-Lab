## Core Claim

A method with K parallel thin blocks at reduced dimensionality connected via learned down/up projections and composed via top-k block routing, using hybrid attention that pairs one shared softmax block for global context with Gated DeltaNet linear attention in routed blocks.

## Limitations

- Scaling sparse block routing to many blocks creates an attention coverage problem, as each block sees fewer tokens.
