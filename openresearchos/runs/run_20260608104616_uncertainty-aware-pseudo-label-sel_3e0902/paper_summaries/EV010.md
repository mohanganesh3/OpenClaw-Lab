# EV010: Annotation-Efficient Active Test-Time Adaptation with Conformal Prediction

## Problem
Existing Active Test-Time Adaptation (ATTA) methods use heuristic uncertainty measures and suffer from low data selection efficiency, wasting human annotation budget during deployment under domain shift.

## Core Claim
CPATTA achieves annotation-efficient ATTA by introducing principled, coverage-guaranteed uncertainty through conformal prediction, consistently outperforming state-of-the-art methods by ~5% in accuracy.

## Key Contributions
- Smoothed conformal scores with top-K certainty measure for fine-grained uncertainty quantification
- Online weight-update algorithm driven by pseudo coverage to maintain valid coverage under domain shift
- Domain-shift detector that dynamically adjusts human annotation budget based on detected shifts
- Staged update scheme balancing human-labeled and model-labeled data with different learning rates

## Method
CPATTA employs two complementary conformal predictors (pretrained and real-time adapted models) to compute top-K certainty scores for sample selection, uses adaptive weighting based on pseudo coverage to maintain coverage guarantees, implements domain-shift detection to adjust annotation budgets, and applies staged updates with human-labeled data prioritized over model-labeled data.
- Components: Smoothed conformal scores with top-K certainty measure, Online weight-update algorithm using pseudo coverage feedback, Domain-shift detector (DSS) for adapting human supervision, Staged update scheme with buffer-specific learning rates
- Key innovation: Adaptive weighting mechanism that uses pseudo coverage as online feedback to self-calibrate conformal predictors under evolving domains, maintaining valid coverage guarantees despite exchangeability violation.
- Assumptions: Availability of small source-domain calibration set for initial conformal predictor setup; Real-time model predictions can serve as reliable pseudo labels for coverage estimation; Domain shifts are gradual enough for pseudo coverage to provide meaningful feedback

## Experimental Setup
- Datasets: PACS (4 domains, 7 categories), VLCS (4 domains, 5 classes), Tiny-ImageNet-C (200 classes, corruption severity 5)
- Baselines: Tent [7], CoTTA [9], CEMA [12], SimATTA [11], EATTA [13]
- Metrics: Real-Time Accuracy, Post-Adaptation Accuracy, Coverage Gap, Annotation Efficiency (EffH, EffM)
- Compute: single GPU

## Results
Headline: CPATTA with α=0.2 achieves 87.13% post-adaptation accuracy on PACS, improving over SimATTA by nearly 9%

- Improvement over baseline: CPATTA achieves roughly 5% higher accuracy than SOTA ATTA methods
