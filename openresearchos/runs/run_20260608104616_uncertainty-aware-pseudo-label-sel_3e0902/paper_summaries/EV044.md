## Problem
Semi-supervised domain adaptation (SSDA) struggles with scarce semantic label information in target domains, limiting performance of existing approaches.

## Core Claim
Graph-based Adaptive Betweenness Clustering (G-ABC) achieves superior categorical domain alignment by constructing and refining a reliable heterogeneous graph for semantic propagation.

## Key Contributions
- Heterogeneous graph construction connecting labeled source/target samples with unlabeled target samples
- Connectivity refinement via Confidence Uncertainty based Node Removal (CUNR) and Prediction Dissimilarity based Edge Pruning (PDEP)
- Adaptive Betweenness Clustering for semantic transfer using pairwise label similarity

## Method
Constructs a heterogeneous graph between labeled source/target samples and unlabeled target samples, refines it using CUNR and PDEP to remove noisy nodes/edges, then applies Adaptive Betweenness Clustering for semantic propagation.
- Components: Graph Construction with pairwise label similarity, Connectivity Refinement (CUNR + PDEP), Adaptive Betweenness Clustering with binary cross-entropy loss
- Key innovation: The integration of confidence-based node removal and prediction dissimilarity-based edge pruning to create a reliable graph structure before semantic propagation, addressing the unreliability of initial graph connectivity.
- Assumptions: Labeled samples across domains have trustworthy ground-truth labels; Features and labels vary smoothly over graph edges; High prediction confidence indicates reliable unlabeled samples
