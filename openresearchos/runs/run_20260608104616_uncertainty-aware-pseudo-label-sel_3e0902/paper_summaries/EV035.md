## Results
Headline: PromptEM achieves 100% F1-score on REL-HETER and REL-TEXT datasets

| Method | Dataset | Metric | Value |
|---|---|---|---|
| PromptEM | REL-HETER | F1 | 100% |
| PromptEM | SEMI-HOMO | F1 | 86.1% |
| PromptEM | SEMI-HETER | F1 | 93.9% |
| PromptEM | SEMI-REL | F1 | 94.0% |
| PromptEM | SEMI-TEXT-c | F1 | 40.8% |
| PromptEM | SEMI-TEXT-w | F1 | 15.7% |
| PromptEM | REL-TEXT | F1 | 26.5% |
| PromptEM | GEO-HETER | F1 | 78.0% |

- Ablations: PromptEM w/o PT: 15.7% average F1-score drop; PromptEM w/o LST: 6.8% improvement on SEMI-TEXT-c; PromptEM w/o DDP: Prunes data while maintaining/improving accuracy
- Improvement over baseline: PromptEM outperforms all baselines across all datasets, achieving SOTA results
