## Problem
Existing pseudo-labeling methods assume labeled data is perfect, but real-world data often contains mislabeling or ambiguity that degrades performance.

## Core Claim
Investigating and improving labeled data quality through data characterization and selection significantly enhances pseudo-labeling performance.

## Key Contributions
- Introduction of DIPS, a novel data characterization and selection framework for pseudo-labeling
- Demonstration of DIPS's effectiveness across various pseudo-labeling methods and real-world tabular datasets
- Showcasing that DIPS improves data efficiency and reduces performance gaps between different pseudo-labelers

## Method
DIPS characterizes and selects useful labeled and pseudo-labeled samples via analysis of learning dynamics, mitigating noise propagation from mislabeled data.
- Components: Data characterization through learning dynamics analysis, Sample selection mechanism for both labeled and pseudo-labeled data
- Key innovation: The use of learning dynamics to identify and select high-quality samples, which helps mitigate the impact of noisy labels in pseudo-labeling by preventing error propagation.
- Assumptions: Labeled data may contain noise (mislabeling, ambiguity); Learning dynamics can reveal sample usefulness; Selected samples will improve model performance

## Experimental Setup
- Datasets: MNIST (100 labeled, 60000 unlabeled), FashionMNIST (100 labeled, 60000 unlabeled), SVHN (100 labeled, 73257 unlabeled), CIFAR-10 (4000 labeled, 45000 unlabeled), CIFAR-100 (4000 labeled, 45000 unlabeled), Tiny-ImageNet (500 labeled, 100000 unlabeled)
- Baselines: Greedy Pseudo-Labeling, UPS, FlexMatch, SLA, CSA, DIPS
- Metrics: Accuracy
- Compute: Single GPU (NVIDIA V100)

## Results
Headline: DIPS achieved 94.2% accuracy on CIFAR-10 with 20% label noise

| Method | Dataset | Metric | Value |
|---|---|---|---|
| DIPS | CIFAR-10 | accuracy | 94.2% |
| FlexMatch | CIFAR-10 | accuracy | 91.8% |
| UPS | CIFAR-10 | accuracy | 89.5% |
| Greedy PL | CIFAR-10 | accuracy | 87.3% |
| DIPS | CIFAR-100 | accuracy | 78.6% |
| FlexMatch | CIFAR-100 | accuracy | 74.2% |
| UPS | CIFAR-100 | accuracy | 71.8% |
| Greedy PL | CIFAR-100 | accuracy | 68.9% |

- Ablations: Confidence threshold component (reduces accuracy by 3.2%); Aleatoric uncertainty component (reduces accuracy by 2.1%); Learning dynamics window size (optimal at 10 epochs)
- Improvement over baseline: DIPS outperforms FlexMatch by 2.4% on CIFAR-10 and 4.4% on CIFAR-100 under 20% label noise
