## Problem
Adapting pre-trained models to new domains without labeled data and without access to source data, which leads to overfitting, underfitting, and poor generalization due to domain discrepancies and noise.

## Core Claim
DRIVE, a dual-model framework leveraging information variability and entropic consistency, improves robustness and generalization in Source-Free Unsupervised Domain Adaptation by capturing diverse target domain characteristics.

## Key Contributions
- A dual-model architecture with parallel models initialized with identical weights to capture diverse target domain characteristics
- An entropy-aware pseudo-labeling strategy that adjusts label weights based on prediction uncertainty
- A two-stage adaptation process where Stage 1 aligns models on stable features using mutual information consistency loss, and Stage 2 dynamically adjusts perturbation levels based on Stage 1 losses

## Method
DRIVE employs a dual-model architecture with one model exposed to PGD perturbations guided by mutual information, using mutual information consistency loss and entropy-aware pseudo-labeling in a two-stage process with dynamic perturbation adjustment.
- Components: Dual-model architecture with parallel models (Model 1: clean samples, Model 2: PGD-perturbed samples), Mutual information consistency loss for aligning model predictions, Entropy-aware pseudo-labeling strategy weighting model outputs by their respective entropies, Dynamic perturbation adjustment mechanism where perturbation magnitude is proportional to Stage 1 losses
- Key innovation: The dual-model approach with mutual information-guided perturbations enables the framework to harness both alignment (through MI consistency) and diversity (through controlled perturbations) during adaptation, improving robustness against target domain uncertainty.
- Assumptions: Source and target domains share common classes; The pre-trained source model can serve as initialization for target model adaptation; Mutual information can effectively guide meaningful perturbations in the target domain

- Ablations: Entropy-Aware Predictor improves performance on A →D and D →W tasks; Perturbed Model Encouragement mechanism exposes models to perturbations via PGD guided by mutual information; Dynamic Perturbation Adjustment mechanism
