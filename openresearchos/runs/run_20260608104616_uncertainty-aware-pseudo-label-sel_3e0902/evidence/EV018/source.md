# EV018: Semi-Supervised UAV-Based Succulent Plants Classification Detection with Efficient Teacher-Student Framework

## Abstract

Accurate classification of succulent plants is vital for applications in intelligent horticultural management. However, traditional methods relying on manual observation or fully supervised learning are limited by high annotation costs and the scarcity of large, high-quality labeled datasets. To address these challenges, this study proposes a semi-supervised learning framework for UAV-based succulent classification. The method first involves fine-tuning a teacher model on a small, labeled dataset, which then generates pseudo-labels for a large volume of unlabeled UAV imagery. These pseudo-labels are iteratively refined to train a student model, effectively reducing dependence on manual annotation while enhancing model generalization. Experimental results show that as unlabeled data was incorporated, the mean Average Precision (mAP@50) increased from 83.1% to 85.5% with negligible impact on inference speed. Among evaluated architectures, YOLOv12x achieved the best overall performance. The proposed framework offers cost-effective solution for large-scale succulent classification in real-world environments.
