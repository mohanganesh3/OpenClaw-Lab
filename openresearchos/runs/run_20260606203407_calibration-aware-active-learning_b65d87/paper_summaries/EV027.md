## Problem
Open-set active learning where models must handle both known and unknown classes while maintaining high query precision from early training stages.

## Core Claim
Their method achieves superior control and stability in purity regulation, maintaining high query precision from early training stages through a novel purity metric and flexible sampling strategy.

## Key Contributions
- Purity-based candidate selection using auxiliary head outputs
- Precision-based candidate refinement with calibrated target query precision
- Information-based final query selection using Jensen-Shannon divergence

## Method
Trains classifier with LCE for primary head and LEDL for auxiliary head, computes purity margins using auxiliary outputs, fits GMM on logit margins to model purity regimes, and performs multi-stage selection (purity → precision → information) to select queries.
- Components: Purity margin computation using auxiliary head outputs, GMM fitting on logit margins to model purity regimes, Multi-stage candidate selection (purity-based, precision-based, information-based)
- Key innovation: The purity metric Spurity(x) = maxc∈Ck oc - maxc∈Cˆu oc which effectively distinguishes between known and unknown classes, enabling precise candidate selection and stable purity regulation.
- Assumptions: The auxiliary head can provide meaningful outputs for both known and unknown classes; Purity regimes can be effectively modeled by a 3-component GMM; High-purity samples provide greater utility when the model is undertrained
