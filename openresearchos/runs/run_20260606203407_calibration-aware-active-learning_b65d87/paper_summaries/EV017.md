## Problem
Existing multi-domain skin lesion segmentation models inadequately utilize prediction uncertainty, leading to blind fusion of artifacts, passive topological reasoning, and overfitting to label noise.

## Core Claim
UGDD-Net achieves superior segmentation reliability and generalization by actively integrating uncertainty throughout the learning lifecycle via an uncertainty-in-the-loop architecture.

## Key Contributions
- Uncertainty-Guided Bi-directional Feature Fusion (UGBFF) module dynamically modulates spatial-spectral interactions using pixel-level uncertainty
- Uncertainty-Guided Graph Refinement (UGGR) module constructs priority-guided topology-aware graphs for active global reasoning
- Uncertainty-Guided Margin-Adaptive Loss (UGML) enforces strict constraints on confident pixels while relaxing penalties on uncertain ones

## Method
UGDD-Net employs a dual-domain architecture with spatial and frequency branches, enhanced by three uncertainty-guided modules that actively utilize prediction uncertainty for feature fusion, graph-based refinement, and loss adaptation.
- Components: Uncertainty-Guided Bi-directional Feature Fusion (UGBFF), Uncertainty-Guided Graph Refinement (UGGR), Uncertainty-Guided Margin-Adaptive Loss (UGML)
- Key innovation: The 'Glance-and-Gaze' forward mechanism that actively propagates uncertainty signals throughout the entire learning lifecycle, enabling reliable features to rectify unreliable ones and confident predictions to guide uncertain regions.
- Assumptions: Pixel-level uncertainty estimates are meaningful and accurately reflect prediction confidence; Dual-domain representations capture complementary spatial and frequency information; Uncertainty can be effectively used to guide both feature fusion and topological reasoning
