## Problem
Humanoid robots face perceptual limitations with narrow fields of view and self-occlusion that constrain effective workspace in unstructured environments.

## Core Claim
OmniDP enables robust large-workspace humanoid manipulation through omnidirectional 3D perception using an end-to-end LiDAR-driven visuomotor policy.

## Key Contributions
- Development of OmniDP, an end-to-end LiDAR-driven 3D visuomotor policy for large-workspace manipulation
- Time-Aware Attention Pooling mechanism for processing sparse and temporally flickering LiDAR point clouds
- Whole-body teleoperation system for efficient demonstration data collection on full-body humanoid robots
- Demonstration of superior performance in large-workspace and cluttered scenarios compared to RGB-D baselines

## Method
OmniDP processes panoramic LiDAR point clouds through a pyramid convolutional encoder enhanced with time-aware attention, producing features that condition a Diffusion Policy decoder for action generation, enabling 360° workspace awareness without base repositioning.
- Components: Panoramic LiDAR perception system (Livox MID-360) providing 360° horizontal and -7° to 52° vertical FOV, Time-Aware Attention Pooling mechanism aggregating point clouds over temporal windows with weighted recent observations, Point cloud preprocessing pipeline cropping points beyond 1.3m and downsampling to 4096 points, Diffusion Policy decoder generating coordinated upper-body actions conditioned on global features and joint states
- Key innovation: Time-Aware Attention Pooling mechanism that stabilizes perception by aggregating sparse LiDAR data over temporal windows while emphasizing recent observations, enabling robust processing of flickering point clouds for policy learning.
- Assumptions: Objects within 1.3m of robot are most relevant for manipulation tasks; Proprioceptive joint states (43-dim) are available for full-body coordination; Lower body is controlled separately by pre-trained HOMIE algorithm; LiDAR point clouds are represented in egocentric frame eliminating calibration needs

## Experimental Setup
- Datasets: Teleoperated LiDAR point clouds + proprioceptive joint positions (29 DoFs)
- Baselines: DP (240x424 RGB images), DP3 (4096-point depth-derived point clouds), iDP3 (4096-point depth-derived point clouds)
- Metrics: Task success rate, Collision rate
- Compute: null

## Results
Headline: OmniDP achieved 82/120 total success rate (16/20 Pick & Place, 12/20 Pour OV, 15/20 Pick & Place real, 12/20 Hand Over OV, 11/20 Wipe OV)
