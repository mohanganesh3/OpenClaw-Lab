## Problem
The inherent sensitivity of thin film growth to hidden parameters such as substrate conditions and chamber conditions can compromise the performance of machine learning models in autonomous thin-film deposition.

## Core Claim
A fully autonomous thin-film deposition system with a calibration layer and active learning can fabricate silver thin films with optical reflected power ratios within 2.5% of the target in an average of 2.3 attempts.

## Key Contributions
- A calibration layer approach to account for hidden parameter variations in thin film deposition
- An autonomous learning system that combines in-situ optical spectroscopy, robotic sample handling, and Gaussian Process Regression models

## Method
The authors developed a fully autonomous physical vapor deposition system that uses a calibration layer to measure and account for hidden parameter variations, combined with Gaussian Process Regression models and an active learning algorithm to optimize the exploration of the parameter space.
- Components: Calibration layer: A thin layer grown at T = 875°C for 1000 seconds, with its R689 (reflectance at 689 nm) denoted as Rc, used to quantify fluctuations in growth dynamics, Gaussian Process Regression models: Two models for R (reflectance) and A (absorbance) with RBF kernels, using input parameters (T, t, λ, Rc), Active learning algorithm: Selects the effusion cell temperature T for growth based on maximizing prediction uncertainty
- Key innovation: The calibration layer approach, which uses a thin layer grown at the beginning of each deposition to measure and account for hidden parameter variations, enabling the ML model to adapt to chamber and substrate condition fluctuations.
- Assumptions: The R689 at 1000 seconds is moderately correlated with variations in optical properties at later times; The exponential relationship between tmax and T (Eqn. 3) accurately represents the relationship between temperature and growth rate; The RBF kernel is effective for modeling the relationships between input parameters and optical properties

## Experimental Setup
- Datasets: silver thin film deposition dataset (20,000+ data points)
- Baselines: Control experiment without calibration layers and autonomous learning

## Results
Headline: Autonomous system achieved target optical properties with MAE of 0.0246 compared to 0.0618 for control experiment

- Ablations: Control experiment without calibration layers and autonomous learning showed significantly worse performance (higher MAE and more attempts per target)
- Improvement over baseline: MAE improved by 0.0372 (from 0.0618 to 0.0246) and average attempts per target reduced by 1.6 (from 3.6 to 2)
