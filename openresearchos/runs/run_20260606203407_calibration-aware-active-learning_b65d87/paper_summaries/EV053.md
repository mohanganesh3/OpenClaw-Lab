## Problem
Detecting out-of-distribution (OOD) samples is critical for safe neural network deployment in safety-critical applications, where existing methods like maximum softmax probability (MSP) lack theoretical grounding and suffer from miscalibration.

## Core Claim
VNDUQE achieves superior novelty detection by leveraging the Deep Variational Information Bottleneck (VIB) to constrain information flow, enabling complementary detection signals via KL divergence (for far-OOD) and prediction entropy (for near-OOD).

## Key Contributions
- Proposes VNDUQE framework using VIB for novelty detection with explicit information constraints
- Demonstrates complementary detection signals: KL divergence achieves 100% AUROC on far-OOD (noise), while entropy achieves 94.7% AUROC on near-OOD (novel digit classes)
- Shows combined detection strategy achieves 95.3% average AUROC and 92% TPR at 5% FPR, a 32 percentage point improvement over MSP baseline

## Method
Trains VIB models on MNIST with held-out classes, using a Gaussian encoder with diagonal covariance and reparameterization trick, evaluating OOD detection through KL divergence and prediction entropy metrics.
- Components: Gaussian encoder p(z|x) = N(µ(x), Σ(x)) with diagonal covariance, VIB objective: L = E[−log q(y|z)] + β·KL[p(z|x)∥N(0, I)], Reparameterization trick: z = µ(x) + σ(x) ⊙ ϵ for gradient optimization, Novelty metrics: KL divergence (Eq. 10) and prediction entropy (Eq. 11)
- Key innovation: Information bottleneck compression (β=10⁻³) forces representations to discard irrelevant input information while preserving task-relevant features, producing fundamentally more reliable uncertainty estimates as evidenced by 38% reduction in Expected Calibration Error.
- Assumptions: MNIST with held-out digit classes effectively simulates realistic OOD scenarios; KL divergence and prediction entropy are complementary signals for OOD detection; Simple decoder architecture forces compression into encoder representations
