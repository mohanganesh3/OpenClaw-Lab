# EV040: Ask-n-Learn: Active Learning via Reliable Gradient Representations for Image Classification

## Problem
Existing active learning methods are limited by non-calibrated uncertainties, insufficient exploration-exploitation trade-offs, and confirmation bias, leading to inefficient use of labeling budgets.

## Core Claim
Ask-n-Learn, an active learning approach using reliable gradient embeddings from calibrated models and data augmentation, significantly outperforms state-of-the-art baselines.

## Key Contributions
- Proposing the use of calibrated predictive models to obtain reliable gradient embeddings for active learning.
- Introducing a data augmentation strategy to reduce confirmation bias during pseudo-labeling.
- Demonstrating consistent improvements over BADGE across multiple benchmarks (CIFAR-10, SVHN, Fashion-MNIST, MNIST).

## Method
Ask-n-Learn builds a calibrated classifier, estimates pseudo-labels for unlabeled data, refines low-confidence pseudo-labels via augmentation, computes gradient embeddings, and selects diverse samples using K-means++ seeding.
- Components: Building calibrated classifiers using VWCC or LWCC objectives., Computing gradient embeddings based on pseudo-labels and cross-entropy loss., Refining pseudo-labels with data augmentation to reduce confirmation bias., Selecting query samples using K-means++ on gradient embeddings for diversity.
- Key innovation: The key innovation is the use of prediction calibration to ensure gradient embeddings are reliable, combined with a data augmentation strategy to produce more trustworthy pseudo-labels, thereby addressing confirmation bias and improving the exploration-exploitation trade-off.
- Assumptions: The model can be effectively calibrated using VWCC or LWCC objectives.; Data augmentation can provide more reliable pseudo-label estimates for low-confidence samples.; Gradient embeddings computed from calibrated pseudo-labels are effective representations for sampling.

## Experimental Setup
- Datasets: MNIST, Fashion-MNIST, CIFAR-10, SVHN
- Baselines: Random, Entropy, Confidence, BADGE
- Metrics: accuracy, ECE, NLL, Brier Score
- Compute: single GPU

## Results
Headline: CIFAR-10: Ask-n-Learn (VWCC) achieves 70% accuracy with 10K labeled samples vs. baselines requiring nearly 20K samples

| Method | Dataset | Metric | Value |
|---|---|---|---|
| Ours (VWCC) | CIFAR-10 | accuracy | 70% |
| Ours (VWCC) | CIFAR-10 | ECE | 0.10 |
| Ours (VWCC) | CIFAR-10 | NLL | 0.4 |
| Ours (VWCC) | CIFAR-10 | Brier Score | 0.3 |
| Ours (LWCC) | SVHN | accuracy | 1%−2% improvement at low query counts |
| BADGE | CIFAR-10 | accuracy | requires 20K samples for 70% |

- Ablations: VWCC vs LWCC: VWCC superior except on SVHN where LWCC converges faster
- Improvement over baseline: Ask-n-Learn achieves same accuracy with 50% fewer labels than baselines on CIFAR-10; 1-2% accuracy gain on SVHN at low query counts; significantly lower ECE/NLL/Brier scores across all datasets

## Critical Reading
- Stated limitations: Non-calibrated uncertainties in existing active learning strategies; Insufficient trade-off between data exploration and exploitation; Presence of confirmation bias in active learning approaches
- Unstated weaknesses: Potential computational complexity of gradient embedding computation; Sensitivity to hyperparameter choices for pseudo-label estimation; Limited evaluation on diverse medical imaging datasets; Possible vulnerability to noise in pseudo-labels
- Threats to validity: Limited evaluation on medical images with unique characteristics; Potential overfitting to specific data distributions; Uncertainty about generalization to rare medical conditions
- Open questions: How does the method perform specifically on medical images with class imbalance?; What is the impact of pseudo-label quality on the overall performance?; How does this method compare to other calibration-aware active learning approaches?; What are the computational requirements for deployment in clinical settings?
