## Problem
Molecular property prediction with small datasets where high-quality experimental data is scarce, addressing limitations of deep learning methods in small data regimes.

## Core Claim
FlowMO enables accurate molecular property prediction with well-calibrated uncertainty estimates using Gaussian Processes, achieving comparable performance to deep learning methods but with superior uncertainty calibration.

## Key Contributions
- Development of FlowMO, an open-source Python library for molecular property prediction with Gaussian Processes
- Implementation of specialized kernels (Sub-sequence String Kernel and Tanimoto Kernel) for handling molecular representations
- Demonstration through computational experiments that FlowMO achieves comparable predictive performance to deep learning methods with superior uncertainty calibration

## Method
FlowMO builds upon GPflow and RDKit to implement Gaussian Processes for molecular property prediction using two specialized kernels: a Sub-sequence String Kernel (SSK) for SMILES string representations and a Tanimoto Kernel (TK) for fingerprint representations, enabling predictions with well-calibrated uncertainty estimates.
- Components: Sub-sequence String Kernel (SSK): Measures similarity between SMILES strings through shared sub-strings, Tanimoto Kernel (TK): Measures similarity between molecules via their fingerprints using Tanimoto similarity, GPU support for efficient computation
- Key innovation: Development of specialized kernels (SSK and TK) that allow Gaussian Processes to effectively handle molecular representations (SMILES strings and fingerprints), which are high-dimensional and not suitable for standard GP kernels designed for low-dimensional continuous spaces.
- Assumptions: Standard GPs are adequate for the datasets considered in the paper; Molecular representations (SMILES strings and fingerprints) can be effectively used as inputs to GPs with the proposed kernels; The datasets are small enough that the computational complexity of GPs is manageable

## Experimental Setup
- Datasets: ESOL, FreeSolv, Photoswitch
- Baselines: GP-SSK, GP-ECFP, BNN, ANP, SMILES-X, DeepChem
- Metrics: RMSE, Calibration score
- Compute: null

## Results
Headline: GP-SSK achieved RMSE of 0.24 on Photoswitch dataset

| Method | Dataset | Metric | Value |
|---|---|---|---|
| GP-SSK | ESOL | RMSE | 0.54 |
| GP-SSK | FreeSolv | RMSE | 0.63 |
| GP-SSK | Photoswitch | RMSE | 0.24 |
| GP-ECFP | ESOL | RMSE | 0.56 |
| GP-ECFP | FreeSolv | RMSE | 0.64 |
| GP-ECFP | Photoswitch | RMSE | 0.25 |
| BNN | ESOL | RMSE | 0.57 |
| BNN | FreeSolv | RMSE | 0.65 |

- Ablations: Comparison of SMILES strings vs ECFP fingerprints; Comparison of SSK vs ECFP features
- Improvement over baseline: GP-SSK improved RMSE by 0.02 over GP-ECFP on ESOL, 0.01 on FreeSolv, and 0.01 on Photoswitch
