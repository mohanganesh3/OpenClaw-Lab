## Problem
How to efficiently learn multiple regression models simultaneously in active learning scenarios, specifically for drivability calibration where all outputs require equal accuracy.

## Core Claim
The proposed CVH strategy reduces measurement points by up to 30% compared to sequential space-filling while outperforming existing active learning methods for multi-output regression.

## Key Contributions
- A new active output selection strategy (CVH) that dynamically selects the leading model based on cross-validation error
- Empirical comparison of CVH against sequential (SQ), round-robin (RR), global (G), and sequential space-filling (SF) strategies across toy examples and benchmark datasets

## Method
The method uses Gaussian Processes to model multiple outputs in the same input space, with CVH selecting the output having the highest filtered cross-validation error as the leading model for query placement in each iteration.
- Components: Gaussian Process regression for multi-output modeling, Normalized root mean squared K-fold cross-validation error (CV10) calculation, Digital moving average filter for error stabilization, Dynamic leadership selection based on highest model error
- Key innovation: Using cross-validation error as the quantitative basis for determining which model should lead the active learning process, rather than heuristic approaches like maximum variance or fixed schedules.
- Assumptions: Measurements are significantly more costly than computation; All outputs share the same input space; Noise levels are within real-world calibration ranges (SNR 7-100); Cross-validation error correlates with model quality needs

## Experimental Setup
- Datasets: Three toy examples with analytical models (sigmoid and polynomial)
- Baselines: SF (Space-Filling), SQ (Sequential Query), RR (Random Regression), CVH (Cross-Validation Hybrid), G (Greedy)
- Metrics: NRMSEval, NRMSEval,Σ, Mean (µ), Standard Deviation (σ)
- Compute: Not specified
