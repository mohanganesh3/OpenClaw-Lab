## Problem
The paper targets the evaluation of machine learning agents on machine learning engineering tasks, specifically predicting molecular properties from molecular structures.

## Core Claim
MLE-bench provides a standardized benchmark framework for evaluating machine learning agents on machine learning engineering tasks with a focus on molecular property prediction.

## Key Contributions
- A comprehensive benchmark framework (MLE-bench) for evaluating ML agents on ML engineering tasks
- A structured evaluation metric (Log of Mean Absolute Error) that normalizes improvements across different scalar coupling types
- A rich dataset with multiple molecular features including XYZ coordinates, dipole moments, shielding tensors, and Mulliken charges

## Method
The method involves training ML agents to predict scalar coupling constants between atom pairs using molecular structure features and additional molecular data, with evaluation based on a log-MAE metric averaged across coupling types.
- Components: Feature extraction from molecular structure files (XYZ coordinates), Integration of additional molecular data (dipole moments, magnetic shielding tensors, Mulliken charges), Log-MAE evaluation metric with floor at 1e-9 for numerical stability
- Key innovation: The key innovation is the Log of Mean Absolute Error metric that ensures equal weight for 1% improvement regardless of scalar coupling type, calculated as the average of log(MAE) across all coupling types.
- Assumptions: Molecular structure files (XYZ) provide sufficient information for accurate property prediction; Additional molecular data (dipole moments, shielding tensors, etc.) are relevant and useful features; Test set molecules are completely unseen during training (molecule-wise split)
