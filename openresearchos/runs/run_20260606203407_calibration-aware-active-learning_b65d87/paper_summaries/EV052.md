# EV052: Active Sequential Posterior Estimation for Sample-Efficient Simulation-Based Inference

## Problem
Simulation-based inference methods require large numbers of samples and struggle with high-dimensional settings, making inference prohibitive for resource-intensive simulators.

## Core Claim
ASNPE improves sample efficiency by integrating active learning into the inference loop to estimate the utility of simulation parameter candidates.

## Key Contributions
- Introduction of active sequential neural posterior estimation (ASNPE) with active learning scheme
- Acquisition function that balances epistemic uncertainty and likelihood in parameter selection
- Demonstration of effectiveness in high-dimensional travel demand calibration problem

## Method
ASNPE sequentially trains neural density estimators while selecting parameters that maximize expected divergence between posterior estimates, using an acquisition function that targets epistemic uncertainty.
- Components: Acquisition function α(θ, p(ϕ|D)) = ˜p(θ) · E[ (p(θ|x₀, D) - p(θ|x₀, ϕ))² ], Sequential training of NDEs via APT with proposal distributions ˜p(r)(θ), Optional MC-dropout for approximating Bayesian model parameter posterior p(ϕ|D)
- Key innovation: Acquisition function that selects parameters maximizing expected squared divergence between marginal and conditional posteriors, targeting epistemic uncertainty in the NDE model.
- Assumptions: NDE can be trained via maximum likelihood using Approximate Posterior Transformation (APT); Proposal distributions connect to prior support covering entire parameter domain; MC-dropout provides consistent sampling for Bayesian posterior approximation (when used)

## Experimental Setup
- Datasets: eight scenario priors p(d; ˆd), common SBI benchmark environments (Bernoulli GLM, SLCP)
- Baselines: SNPE-C (or APT), Approximate Bayesian Computation (ABC), PC-SPSA, non-active SNPE
- Metrics: Root mean squared normalized error (RMSNE)
- Compute: not specified

## Results
Headline: ASNPE outperforms other methods across most settings with better RMSNE scores using fewer simulations

- Ablations: —
- Improvement over baseline: ASNPE reaches better RMSNE scores than SNPE and ABC with fewer simulations

## Critical Reading
- Stated limitations: Many simulation-based inference methods require large numbers of simulation samples; Methods face difficulty scaling to high-dimensional settings; Inference can be prohibitive under resource-intensive simulators
- Unstated weaknesses: Method designed for simulation-based inference may not translate directly to medical image classification where forward model is neural network; May not address specific challenges of medical imaging like class imbalance and domain shifts; Unclear how method handles interpretability requirements in medical applications
- Threats to validity: Domain mismatch between simulation-based inference and medical imaging applications; Potential lack of calibration-specific metrics in evaluation; Unclear performance on medical image classification tasks
- Open questions: How would ASNPE perform on medical image classification tasks?; Can the active learning strategy be adapted for medical imaging challenges?; How does the method ensure calibration of posterior estimates in medical applications?
