## Problem
Language agents struggle to learn quickly and efficiently from trial-and-error as traditional reinforcement learning methods require extensive training samples and expensive model fine-tuning.

## Core Claim
Reflexion, a framework that reinforces language agents through linguistic feedback rather than updating weights, significantly improves agent performance across diverse tasks.

## Key Contributions
- A novel framework called Reflexion that uses verbal reinforcement learning to improve language agents
- Significant performance improvements across diverse tasks (sequential decision-making, coding, language reasoning)
- Ablation and analysis studies on different feedback signals, feedback incorporation methods, and agent types

## Method
Reflexion is formalized as an iterative optimization process where agents (Actor, Evaluator, and Self-Reflection models) work together through trials until the task is completed. After each trial, the Self-Reflection model analyzes the trajectory and reward to produce a summary stored in memory, which is then used to improve future decision-making.
- Components: Actor: Produces trajectories by interacting with the environment, Evaluator: Produces scores for trajectories, Self-Reflection model: Analyzes trajectories and rewards to produce summaries, Episodic memory buffer: Stores reflective text for future reference
- Key innovation: The use of verbal reinforcement learning where agents verbally reflect on task feedback signals and maintain their own reflective text in an episodic memory buffer to induce better decision-making in subsequent trials, rather than updating model weights.
- Assumptions: The agent has access to external environments that provide feedback signals; The feedback signals can be either scalar values or free-form language; The agent can store and retrieve experiences from memory; The memory can be bounded to adhere to LLM context limitations

## Critical Reading
- Stated limitations: —
- Unstated weaknesses: The paper focuses on language agents for general environments, not specifically medical domains; No discussion of how verbal reflection translates to visual/spatial reasoning needed for medical images; Lack of evaluation on high-stakes domains where calibration is critical; No comparison with traditional active learning methods; Potential for reflection to reinforce biases in medical decision-making
- Threats to validity: Generalizability from abstract environments to complex medical image classification is uncertain; Effectiveness of linguistic feedback for visual tasks is unproven; Potential for reflection to amplify existing model biases in medical contexts; Lack of theoretical grounding for how verbal reflection improves calibration
- Open questions: How would the verbal reflection mechanism adapt to visual medical data?; Can the reflection process provide calibrated uncertainty estimates?; How does this approach compare to uncertainty-based active learning in medical imaging?; What modifications are needed to handle the high-stakes nature of medical decisions?
