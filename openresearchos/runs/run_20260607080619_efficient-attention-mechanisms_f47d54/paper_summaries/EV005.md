## Problem
Existing AI model development agents are limited by static, outdated, and sparse parametric knowledge in underlying LLMs, hindering their ability to build high-performing models for scientific discovery.

## Core Claim
AIBuildAI-2, a knowledge-enhanced agent with an evolving hierarchical knowledge system, can autonomously build AI models at a level comparable to expert human practitioners.

## Key Contributions
- Introduction of AIBuildAI-2, a knowledge-enhanced agent for autonomous AI model development.
- Design of a hierarchical, evolving knowledge system that grounds decisions in verifiable expertise.
- Demonstration of state-of-the-art performance on MLE-Bench (70.7% medal rate) and top 6.6% ranking in a real-world Kaggle heart disease competition.

## Method
AIBuildAI-2 uses a hierarchical knowledge system with high-level instructions and low-level documents. The agent dynamically loads relevant context for each task, grounds decisions in external knowledge, and distills completed runs into new knowledge to evolve the system.
- Components: Hierarchical Knowledge System: Organizes curated AI development knowledge into topical categories (high-level instructions) and sub-documents (low-level details)., Dynamic Context Loading: Agent loads only task-relevant knowledge context based on its current state., Knowledge Evolution: Distills each completed AI task run into structured takeaways and writes them back into the knowledge system., Sub-agents: Utilizes a knowledge-enhanced hierarchical sub-agent design for autonomous development.
- Key innovation: The continual evolution of the external knowledge system by distilling completed task experiences into structured knowledge, enabling the agent to overcome the limitations of static LLM parametric knowledge and ground decisions in verifiable expertise.
- Assumptions: The agent can reliably extract structured takeaways from completed task runs.; The initial curated knowledge hierarchy covers relevant domains for AI model development.; The dynamic loading mechanism can select the most relevant knowledge context for a given task.
