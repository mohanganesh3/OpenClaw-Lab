## Problem
Building generally capable embodied agents that can continuously explore, plan, and develop new skills in open-ended worlds without human intervention.

## Core Claim
VOYAGER is the first LLM-powered embodied lifelong learning agent in Minecraft that can continuously explore the world, acquire diverse skills, and make novel discoveries without human intervention.

## Key Contributions
- An automatic curriculum that maximizes exploration
- An ever-growing skill library of executable code for storing and retrieving complex behaviors
- A new iterative prompting mechanism that incorporates environment feedback, execution errors, and self-verification for program improvement

## Method
VOYAGER consists of three key components: an automatic curriculum for open-ended exploration, a skill library for increasingly complex behaviors, and an iterative prompting mechanism that uses code as action space. The agent interacts with GPT-4 via blackbox queries and uses Mineflayer as the low-level controller.
- Components: Automatic curriculum that maximizes exploration, Skill library for storing and retrieving complex behaviors, Iterative prompting mechanism that incorporates environment feedback, execution errors, and self-verification
- Key innovation: The iterative prompting mechanism that incorporates environment feedback, execution errors, and self-verification for program improvement, allowing the agent to refine its skills over time without human intervention.
- Assumptions: GPT-4 provides sufficient code generation capabilities (5.7× more unique items than GPT-3.5); The Minecraft environment can be adequately represented through the provided control primitive APIs; The agent can make meaningful progress through self-driven exploration
