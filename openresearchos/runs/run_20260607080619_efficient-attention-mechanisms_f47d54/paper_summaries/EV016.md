## Problem
Standard self-attention lacks explicit temporal dynamics, with tokens attending independently based on static feature similarity rather than consistent state evolution over time.

## Core Claim
Self-attention can be reformulated as precision-weighted state estimation under a learned linear stochastic dynamical prior, where attention weights are determined by consistency with probabilistic state predictions.

## Key Contributions
- Introduction of Robust Filter Attention (RFA) as a formulation of self-attention as robust state estimation
- Development of Spectrally Coupled RFA (SC-RFA) to address phase wrap-around in long-range extrapolation
- Dynamical interpretation of positional mechanisms connecting rotational embeddings to transport and uncertainty propagation

## Method
RFA models each token as a noisy observation of a latent trajectory governed by a linear stochastic differential equation. Past keys are propagated to the query position under learned dynamics, with attention weights computed from prediction errors measured under precisions determined by the Differential Lyapunov Equation (DLE).
- Components: Linear stochastic differential equation dynamics for state propagation, Differential Lyapunov Equation (DLE) for precision calculation, Power-law robust weighting for attention computation, Spectrally coupled decay rates in SC-RFA variant
- Key innovation: Attention as precision-weighted state estimation under a stochastic dynamical prior, coupling transport and uncertainty propagation in a principled way where tokens are weighted according to their consistency with learned dynamics.
- Assumptions: Isotropic noise and decay assumptions; Linear stochastic dynamics for state evolution; Gaussian noise models for observations; Fixed training context window (512 tokens)
