## Problem
Accelerating MCMC sampling from complex probability distributions known only up to a normalization constant, particularly for multimodal distributions where traditional methods struggle with mode exploration.

## Core Claim
The proposed Markovian Flow Matching (MFM) algorithm accelerates MCMC by adaptively learning continuous normalizing flows on-the-fly to provide non-local transition kernels, achieving state-of-the-art performance at lower computational cost.

## Key Contributions
- Adaptive MCMC algorithm combining local Markov kernels with flow-informed non-local kernels
- On-the-fly adaptation of continuous normalizing flows using chain samples to specify probability paths
- Adaptive tempering mechanism for discovering multiple modes in target distributions
- Theoretical convergence guarantee to local optima of the flow matching objective

## Method
The method trains a continuous normalizing flow to model the vector field between reference and target distributions using flow matching, where the probability path is specified adaptively using samples from the Markov chain. This learned CNF provides a non-local transition kernel combined with local MCMC kernels in an adaptive scheme.
- Components: Continuous normalizing flow (CNF) trained via flow matching objective, Local Markov transition kernel (e.g., MALA), Non-local flow-informed transition kernel, Adaptive tempering mechanism, On-the-fly adaptation using chain samples
- Key innovation: The adaptive flow matching objective that uses samples from the current Markov chain to define the probability path, allowing the CNF to learn the target distribution structure dynamically during sampling.
- Assumptions: Mild regularity conditions on the vector field (Assumptions B.1-B.6); Bounded parameter sequence that visits compact subsets infinitely often; Continuously differentiable target density function
