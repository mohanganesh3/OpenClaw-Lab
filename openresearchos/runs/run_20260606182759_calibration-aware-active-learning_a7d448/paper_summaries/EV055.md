## Problem
The complexity of explicit construction problems for objects whose existence follows from the probabilistic method, specifically identifying the natural complexity class for such problems.

## Core Claim
APEPP is the natural complexity class for explicit constructions derived from the probabilistic method, with Hard Truth Table being APEPP-complete under PNP reductions.

## Key Contributions
- Placing multiple explicit construction problems into APEPP, including Hard Truth Table and Complexity.
- Showing Hard Truth Table reduces to Empty (APEPP-complete problem) via a circuit encoding construction.
- Proving that derandomizing Shannon's proof implies generic derandomization of the probabilistic method.
- Establishing a corollary: EXPNP contains a language of circuit complexity 2n^Ω(1) iff it contains one of circuit complexity 2^n.

## Method
Reduces explicit construction problems to the Empty problem by constructing circuits that map encodings of 'structured' objects (those failing the property) to their representations, ensuring any string outside the range must satisfy the desired property.
- Components: Empty problem (finding a string outside a circuit's range with m > n outputs), Efficient circuit encoding (using 2s log s + O(s) bits for size-s circuits), Reduction framework leveraging the 'structure' of failing strings (e.g., low-weight or small-circuit strings)
- Key innovation: Using circuit encoding to map all 'non-hard' strings (those with small circuits) into a range smaller than the output space, ensuring any string outside this range must be hard, thus reducing Hard Truth Table to Empty.
- Assumptions: Reductions valid for sufficiently large n (brute-force handles fixed lengths); Circuit lower bounds hold 'almost everywhere' (for all but finitely many n); Empty is total due to the Dual Pigeonhole Principle
