## Problem
Existing multi-domain architectures for skin lesion segmentation inadequately utilize prediction uncertainty, leading to blind fusion of cross-domain artifacts, passive topological reasoning, and rigid optimization that overfits to label noise.

## Core Claim
UGDD-Net transforms uncertainty into an active guiding signal throughout the entire learning lifecycle, enabling more reliable skin lesion segmentation through uncertainty-in-the-loop design.

## Key Contributions
- Uncertainty-Guided Dual-Domain Network (UGDD-Net) with unified architecture
- Novel two-pass 'Glance-and-Gaze' forward mechanism
- Three uncertainty-guided modules: UGBFF, UGGR, and UGML

## Method
UGDD-Net employs a two-pass forward mechanism where the first pass generates initial predictions and uncertainty estimates, and the second pass refines features using uncertainty-guided fusion, graph-based refinement, and adaptive loss weighting.
- Components: Uncertainty-Guided Bi-directional Feature Fusion (UGBFF) module, Uncertainty-Guided Graph Refinement (UGGR) module, Uncertainty-Guided Margin-Adaptive Loss (UGML) module
- Key innovation: The 'Glance-and-Gaze' mechanism that actively uses pixel-level uncertainty to dynamically modulate spatial-spectral interactions and guide global reasoning through priority-based graph propagation.
- Assumptions: Prediction uncertainty can be reliably estimated from model outputs; Uncertainty correlates with segmentation difficulty and domain shift; Domain-invariant structural features can be extracted through frequency domain processing

## Experimental Setup
