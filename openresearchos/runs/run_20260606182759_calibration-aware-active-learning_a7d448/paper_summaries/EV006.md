# EV006: Reflexion: Language Agents with Verbal Reinforcement Learning

## Problem
Large language model agents struggle to efficiently learn from trial-and-error as traditional reinforcement learning requires extensive training samples and expensive model fine-tuning.

## Core Claim
Reflexion, a framework using linguistic feedback rather than weight updates, can significantly improve language agent performance across diverse tasks.

## Key Contributions
- Novel Reflexion framework using verbal reinforcement learning for language agents
- Demonstrated significant performance improvements on benchmarks (91% pass@1 on HumanEval vs GPT-4's 80%)
- Comprehensive ablation studies analyzing feedback signals, incorporation methods, and agent types

## Method
Reflexion implements an iterative optimization process where agents verbally reflect on task feedback signals, maintain reflective text in an episodic memory buffer, and use this memory to induce better decision-making in subsequent trials.
- Components: Actor: produces trajectories by interacting with environment, Evaluator: computes scalar reward scores based on task performance, Self-Reflection model: analyzes trajectories and scores to produce verbal feedback
- Key innovation: Episodic memory buffer storing reflective text that works as 'self-hints' for future decision-making, enabling agents to learn from mistakes without weight updates.
- Assumptions: Feedback signals can be scalar values or free-form language; Memory buffer can be bounded (typically 1-3 experiences) due to LLM context limitations; Agents can access external environments and receive task feedback

## Experimental Setup
- Datasets: HumanEval Python, HotPotQA (100 samples)
- Baselines: Baseline without Reflexion, CoT (GT), ReAct
- Metrics: Pass@1 accuracy
- Compute: GPU

## Critical Reading
- Stated limitations: The framework focuses on language agents rather than specifically addressing calibration issues; Limited information about generalization to specialized domains like medical imaging
- Unstated weaknesses: No clear mechanism for applying verbal reflection to image-based tasks; Unclear how episodic memory would handle large medical imaging datasets; No comparison to traditional active learning methods for sample efficiency
- Threats to validity: Effectiveness in specialized domains like medical imaging is unproven; Potential overfitting to specific feedback signal types; Verbal reflection may not capture nuanced feedback needed for image classification
- Open questions: How would verbal reflection work for image-based tasks?; Can the framework be adapted for medical image classification challenges?; How does this approach compare to existing active learning methods in terms of calibration?
