## Problem
Stochastic nonconvex optimization with inexact gradient and Hessian evaluations, extending deterministic adaptive cubic regularization to handle probabilistic accuracy without variance restrictions.

## Core Claim
The stochastic adaptive cubic regularization method achieves optimal O(ε^{-3/2}) expected iteration complexity for reaching first-order stationary points under inexact derivative evaluations.

## Key Contributions
- Adapts deterministic ARC with dynamic Hessian accuracy to stochastic setting with inexact gradients
- Maintains optimal worst-case complexity under probabilistic accuracy assumptions
- Demonstrates computational benefits via numerical tests on finite-sum minimization

## Method
Implements stochastic ARC with probabilistic model accuracy requirements, adaptive Hessian accuracy via Ck, and implicit gradient accuracy conditions, using trust-region style acceptance criteria to ensure complexity preservation.
- Components: Probabilistic accuracy model (Definition 2.1) with events I(1)k, I(2)k, I(3)k, Dynamic Hessian accuracy via adaptive Ck in model (2.2), Implicit gradient accuracy condition (2.10) with bound κg, Acceptance ratio ρk and step size control
- Key innovation: Adaptive accuracy mechanism that adjusts Hessian requirements based on gradient magnitude (via Ck) while maintaining implicit gradient accuracy, enabling optimal complexity without variance constraints.
- Assumptions: Lipschitz continuous Hessian (Assumption 2.1 ii); Bounded inexact derivatives: ∥∇f(xk)∥≤κg, ∥∇2f(xk)∥≤κB (2.12); Probabilistic accuracy: Pr(Ik|F_{k-1})≥p for fixed p∈(0,1] (2.13); Parameter constraints: c≥α(1-β)κg, β∈(0,1), η∈(0,1)
