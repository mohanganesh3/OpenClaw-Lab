## Problem
Building generally capable embodied agents that continuously explore, plan, and develop new skills in open-ended worlds without human intervention.

## Core Claim
VOYAGER is the first LLM-powered embodied lifelong learning agent that achieves open-ended exploration, skill acquisition, and novel discoveries in Minecraft without human intervention.

## Key Contributions
- Automatic curriculum maximizing exploration
- Ever-growing skill library of executable code
- Iterative prompting mechanism with environment feedback, execution errors, and self-verification

## Method
VOYAGER uses GPT-4 via blackbox queries to generate executable code for actions in Minecraft, guided by an automatic curriculum, storing skills in a library, and iteratively refining programs using environment feedback and self-verification.
- Components: Automatic curriculum for exploration, Skill library for storing/retrieving complex behaviors, Iterative prompting mechanism with feedback loops
- Key innovation: The iterative prompting mechanism integrating environment feedback, execution errors, and self-verification to improve code generation and decide task progression.
- Assumptions: Access to GPT-4 API via blackbox queries; Text-only interaction (no visual perception); Mineflayer as low-level controller; Minecraft's game mechanics are learnable
