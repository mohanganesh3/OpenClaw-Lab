## Problem
AI model development remains heavily manual and inaccessible to non-specialists, with existing automatic agents limited by static, outdated knowledge of their underlying LLMs.

## Core Claim
AIBuildAI-2's knowledge-enhanced hierarchical agent with an evolving external knowledge system can autonomously build AI models matching expert human performance.

## Key Contributions
- Hierarchical knowledge system organizing AI development knowledge into high-level instructions and low-level documents
- Continual evolution mechanism distilling completed runs into structured knowledge updates
- Demonstrated state-of-the-art performance on MLE-Bench (70.7% medal rate) and top-tier human expert level in real competitions

## Method
AIBuildAI-2 uses a hierarchical sub-agent architecture with dynamically loaded context from a curated knowledge system, continually updated through experience distillation.
- Components: Hierarchical knowledge system with topical categories and documents, Dynamic context loading based on current task state, Experience distillation mechanism writing structured takeaways back to knowledge system, Sub-agents for autonomous model development tasks
- Key innovation: Continual evolution of external knowledge system through experience distillation, enabling the agent to learn from its own successes and failures rather than relying solely on pre-trained knowledge
- Assumptions: Web-collected AI development documents can be effectively organized into hierarchical categories; Experience distillation can capture valuable engineering insights; Relevant knowledge can be dynamically loaded based on task context
