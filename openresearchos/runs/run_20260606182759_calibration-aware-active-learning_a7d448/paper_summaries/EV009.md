# EV009: AgentBench: Evaluating LLMs as Agents

## Problem
The need for a comprehensive benchmark to evaluate LLMs as agents in interactive environments, revealing performance gaps between commercial and open-source models.

## Core Claim
AGENTBENCH provides a standardized, multi-dimensional benchmark that holistically assesses LLM agent capabilities, exposing critical weaknesses in reasoning and instruction following.

## Key Contributions
- Introduction of AGENTBENCH with 8 distinct interactive environments (OS, Database, Knowledge, Graph, Digital Card Game, Lateral Thinking Puzzle, House-keeping, Web Browsing/Shopping)
- Extensive evaluation of 29 API-based and open-source LLMs, revealing significant performance disparities (commercial avg: 4.01 vs OSS avg: 0.51)
- Identification of failure reasons: poor long-term reasoning, decision-making, and instruction following abilities

## Method
The method constructs 8 interactive environments simulating real-world tasks, evaluates LLMs through task completion metrics, and analyzes failure patterns across reasoning, decision-making, and instruction-following dimensions.
- Components: Environment design with specific interaction protocols (e.g., OS command execution, database queries, web navigation), Evaluation framework measuring task success rates and error categorization, Comparative analysis across 29 models including API-based (GPT-4, Claude-3) and open-source (Llama-2-70B, Vicuna-13B)
- Key innovation: The multi-environment benchmark architecture that simultaneously tests diverse agent capabilities (reasoning, decision-making, instruction following) in interactive settings, revealing systemic weaknesses.
- Assumptions: LLMs can function as autonomous agents in structured environments; Performance in benchmark environments correlates with real-world agent utility; Task completion rates are valid proxies for agent competence

## Experimental Setup
- Datasets: FREEBASE (GrailQA, ComplexWebQuestions, GraphQuestions) - 1,663 questions
- Baselines: gpt-4, gpt-3.5-turbo
- Metrics: F1 score, Exact Match, Executability
