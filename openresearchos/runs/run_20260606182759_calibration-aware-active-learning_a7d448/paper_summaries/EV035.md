## Problem

## Core Claim
This paper provides a systematic review and comparison of several representative approaches spanning classical nonparametric methods and modern generative models for conditional distribution estimation.

## Key Contributions

## Method
The paper reviews and compares several methods for conditional distribution estimation, including the single-index method of Hall and Yao (2005), basis-expansion approaches like FlexCode and DeepCDE, and generative methods like GCDS and conditional DDPM.
- Components: Single-index method using dimension-reducing index and nonparametric smoothing, Basis-expansion approaches converting conditional density estimation to regression problems, Generative approaches using deep generative architectures (GCDS and DDPM)
- Key innovation: The key innovation is the unified evaluation framework for comparing these diverse approaches, ensuring fairness and reproducibility.
- Assumptions: For basis-expansion approaches: conditional density admits well-behaved orthogonal series representation; For generative approaches: appropriate latent noise dimension selection; For single-index method: dimension-reducing index exists
