## Problem
Efficient computation of temporal betweenness centrality for large temporal graphs where exact methods like Brandes' algorithm become impractical.

## Core Claim
MANTRA provides a probabilistic framework for high-quality temporal betweenness centrality approximation with guaranteed accuracy and improved efficiency.

## Key Contributions
- Sample-complexity analysis for temporal betweenness centrality approximation
- Progressive sampling approach using Monte Carlo Empirical Rademacher Averages
- Efficient algorithm for approximating temporal diameter and other graph characteristics

## Method
MANTRA uses a two-phase approach: bootstrap phase to estimate parameters (ˆv and ρ(⋆)) and determine initial sample size, followed by progressive sampling with geometric progression and Rademacher vectors to update estimates until desired accuracy is achieved.
- Components: Bootstrap phase with (⋆)-TBFS visits to estimate variance and path length, Estimation phase with progressive sampling and geometric progression, Update mechanism for betweenness estimates, wimpy variances, and Rademacher vectors
- Key innovation: Leverages negative correlation among nodes and bounded variance to reduce sample size compared to worst-case VC-dimension bounds, enabling ˜O(log n/ε² · |E|) runtime for characteristic approximations.
- Assumptions: Maximum variance of estimators bounded by ˆv ∈ (0, 1/4]; Sum of centralities constrained by ρ(⋆) (average internal nodes in paths); Temporal connectivity rate ζ(⋆) affects approximation quality; Temporal diameter D(⋆) bounds ρ(⋆)

## Experimental Setup
- Datasets: Prefix Foremost, Shortest, public transport networks (25 cities)
- Baselines: ONBRA, exact algorithm
- Metrics: running time (seconds), sample size, memory usage (GB), supremum deviation (error)
- Compute: Single laptop (memory measured in GB; ONBRA required >1TB for some datasets)
