## Problem
Counterfactual Explanations face challenges of stability, synthesizing multiple explanations, plausibility/sparsity guarantees, and practical implementation issues where prescribed recourses are not followed exactly in noisy environments.

## Core Claim
We propose a probabilistic framework that provides sparse local counterfactual rules for each observation, giving rules that provide a range of values capable of changing decisions with high probability, and aggregate these into regional rules for subgroups.

## Key Contributions
- A probabilistic framework for sparse local counterfactual rules that provide ranges of values rather than exact counterfactuals
- Regional counterfactual rules that aggregate local rules to identify shared recourses for subgroups
- Derivation from Random Forest algorithm with statistical guarantees and fidelity to data distribution by selecting recourses in high-density regions

## Method
The paper proposes a probabilistic framework that gives sparse local counterfactual rules for each observation, providing rules that give a range of values capable of changing decisions with high probability. These rules serve as a summary of diverse counterfactual explanations and yield robust recourses. They further aggregate these local rules into regional counterfactual rules, identifying shared recourses for subgroups of the data. The rules are derived from the Random Forest algorithm, which offers statistical guarantees and fidelity to data distribution by selecting recourses in high-density regions.
- Components: Local Counterfactual Rules (L-CR) that provide hyperrectangles for individual observations, Regional Counterfactual Rules (R-CR) that aggregate local rules for subgroups, Projected Forest algorithm for estimating conditional probabilities, Regional Random Forest for probability estimation
- Key innovation: The key innovation is the use of probabilistic rules that provide ranges of values rather than exact counterfactuals, derived from Random Forest with statistical guarantees, which addresses the practical issue where prescribed counterfactual recourses are often not implemented exactly by individuals.
- Assumptions: The use of Random Forest algorithm for deriving the rules; The rules can be derived from high-density regions of the data distribution; The minimal divergent explanation can be identified
