## Problem
How to effectively select informative samples in open-set active learning when unlabeled data contains both known and unknown classes

## Core Claim
The proposed purity-based sampling strategy with GMM modeling achieves superior query precision and test accuracy in open-set active learning

## Key Contributions
- Purity-based candidate selection mechanism that distinguishes between known and unknown samples
- GMM modeling of purity regimes to handle different sample types (high, low, intermediate purity)
- Flexible sampling strategy that maintains high query precision from early training stages

## Method
Trains a dual-head classifier with LCE and LEDL losses, estimates unknown classes, computes purity margins, models purity regimes with GMM, and selects samples based on likelihood and information scores
- Components: Dual-head classifier training (LCE for primary, LEDL for auxiliary), Unknown class estimation using labeled pool and upper bound, Purity margin computation with/without unknown classes, GMM modeling of purity regimes, Precision-based candidate refinement, Information-based final query selection
- Key innovation: The purity-based candidate selection mechanism that computes Spurity(x) = maxc∈Ck oc − maxc∈Cˆu oc for samples with unknown classes, enabling effective discrimination between known and unknown samples
- Assumptions: Labeled pool contains only known classes; Unlabeled pool may contain both known and unknown classes; Number of unknown classes can be estimated from labeled data
