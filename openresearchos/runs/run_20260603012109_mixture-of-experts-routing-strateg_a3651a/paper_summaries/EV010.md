## Core Claim

We propose Graph-Attentive Policy Fusion (GAPF), a lightweight meta-learning framework that fuses frozen MARL expert policies to create an energy-efficient, low-latency routing algorithm for disaster monitoring.

## Limitations

- Existing MARL approaches ignore network topology.
- Existing MARL approaches suffer from memory explosion at scale.
