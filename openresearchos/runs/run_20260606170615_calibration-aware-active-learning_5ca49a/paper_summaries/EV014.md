## Problem
Current machine learning potentials (MLPs) struggle with uncertainty quantification, limiting reliability for active learning, calibration, and out-of-distribution detection.

## Core Claim
Bayesian E(3)-equivariant MLPs with iterative restratification of many-body message passing using joint energy-force NLL loss achieve competitive accuracy while enabling uncertainty-guided applications.

## Key Contributions
- Development of joint energy-force negative log-likelihood (NLLJEF) loss function for explicit uncertainty modeling in both energies and forces
- Systematic benchmarking of multiple Bayesian approaches (deep ensembles, SWAG, IVON, Laplace) across UQ tasks including calibration, OOD detection, and active learning

## Method
The framework employs Bayesian E(3)-equivariant neural networks with iterative restratification of many-body message passing, trained using NLLJEF loss to simultaneously optimize energy/force predictions and their uncertainties, evaluated through unified validation metrics.
- Components: E(3)-equivariant message passing architecture with iterative restratification, Joint energy-force uncertainty modeling via NLLJEF loss, Multiple Bayesian inference methods (DE, SWAG, IVON, Laplace) for uncertainty quantification
- Key innovation: NLLJEF loss function that incorporates force uncertainties into the optimization objective, reducing energy errors by 3-4× and force errors by 2-3× compared to energy-only uncertainty approaches while maintaining competitive accuracy.
- Assumptions: Joint energy-force uncertainty modeling is superior to energy-only approaches; Uncertainty estimates are crucial for practical applications like active learning; Diverse Bayesian methods can be fairly compared through unified metrics
