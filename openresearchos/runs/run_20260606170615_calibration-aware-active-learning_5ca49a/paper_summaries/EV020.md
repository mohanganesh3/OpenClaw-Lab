## Problem
Deterministic embeddings from standard vision-language models struggle to capture uncertainties arising from ambiguities in visual/textual descriptions and multiple possible correspondences between modalities.

## Core Claim
GroVE achieves state-of-the-art uncertainty calibration by obtaining probabilistic embeddings from frozen VLMs through a post-hoc Gaussian Process Latent Variable Model approach.

## Key Contributions
- GroVE framework for post-hoc probabilistic embeddings from frozen VLMs
- GPLVM-based shared latent space learning with single-modal reconstruction and cross-modal alignment objectives
- Demonstrated superior uncertainty calibration across cross-modal retrieval, VQA, and active learning tasks

## Method
GroVE builds a shared low-dimensional latent space using GPLVM where image and text inputs map to unified representations, optimized through single-modal embedding reconstruction and cross-modal alignment objectives to generate uncertainty-aware probabilistic embeddings.
- Components: Gaussian Process Latent Variable Model (GPLVM) framework, Single-modal embedding reconstruction objective, Cross-modal alignment objective
- Key innovation: Leveraging GPLVM's predictive distribution p(g*) = ∫ p(g*|x*)p(x*|z*, zM)dx* to capture uncertainty from both latent space uncertainty and inherent noise, enabling uncertainty quantification without VLM retraining.
- Assumptions: Frozen VLM embeddings provide meaningful representations for latent space learning; Low-dimensional latent space (Q=5 or 10) sufficiently captures cross-modal relationships; GP hyperparameters (inducing points, trade-off parameters) can be tuned via grid search
