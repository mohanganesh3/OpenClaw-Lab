## Core Claim

We present a new meta-learning algorithm, Model-Agnostic Meta-Learning (MAML), that learns a model initialization which can be fine-tuned to new tasks with a single gradient step.

## Limitations

- Optimized for single-step adaptation, which may not be sufficient for more complex tasks.
