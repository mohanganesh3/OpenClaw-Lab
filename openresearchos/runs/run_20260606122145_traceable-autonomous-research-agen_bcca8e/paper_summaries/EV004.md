# EV004: MLE-bench: Evaluating Machine Learning Agents on Machine Learning Engineering

Year: 2024
URL: https://arxiv.org/abs/2410.07095
Full Text: arxiv_abstract_openclaw
Datasets: Kaggle competitions
Baselines: Human baselines from Kaggle leaderboards, Various frontier language models
Best Result: OpenAI's o1-preview with AIDE scaffolding achieves at least the level of a Kaggle bronze medal in 16.9% of competitions
Code: yes

## Core Claim

Evaluation of machine learning agents on machine learning engineering tasks using Kaggle competitions as benchmarks, with human baselines established from Kaggle leaderboards and evaluation of frontier language models using open-source agent scaffolds.
