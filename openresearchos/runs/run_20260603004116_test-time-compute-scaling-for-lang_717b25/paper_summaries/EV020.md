## Core Claim

The method uses a Gaussian-mixture-model-based preprocessing pipeline to extract segmented trajectories from noisy AIS data, then constructs a global knowledge graph with speed distributions stratified by vessel type, time of travel, and direction of travel for travel-time prediction.

## Limitations

- Performance degrades on external test sets compared to held-out test sets
- Method relies on AIS data, which may have gaps or inaccuracies
- System requires a principled fallback for cases where historical statistics are insufficient
