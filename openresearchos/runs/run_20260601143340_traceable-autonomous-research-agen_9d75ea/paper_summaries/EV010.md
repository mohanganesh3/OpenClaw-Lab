# EV010: SWE-agent: Agent-Computer Interfaces Enable Automated Software Engineering

Year: 2024

URL: https://arxiv.org/abs/2405.15793

## Core Claim

Demonstrates how agents can operate computer interfaces to solve software engineering tasks, with relevance to local experiment execution and reproducible command logs.

## Method Or Focus

- Agent-computer interfaces
- Automated software engineering
- Local experiment execution
- Reproducible command logs

## Limitations To Track

- Potential limitations in scaling to distributed environments
- Challenges in ensuring reproducibility across different environments

## Possible Gap

The paper focuses on agents operating computer interfaces for software engineering tasks, leaving open questions about how these agents can handle more complex, multi-step software engineering workflows or adapt to new programming languages and frameworks.
