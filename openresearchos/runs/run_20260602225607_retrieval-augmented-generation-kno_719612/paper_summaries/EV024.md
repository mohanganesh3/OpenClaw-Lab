## Core Claim

We present an egocentric vision pipeline for the Unitree G1 humanoid that retrieves task- and context-appropriate impedance parameters via a Vision-Language Model (VLM) with Retrieval-Augmented Generation (RAG).

## Limitations

- Method is demonstrated on a specific robot model (Unitree G1), which may limit generalizability.
