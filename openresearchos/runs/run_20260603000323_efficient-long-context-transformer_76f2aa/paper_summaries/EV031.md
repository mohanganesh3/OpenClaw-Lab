## Core Claim

This paper proposes a low-dose CT denoising network using an adaptive global context attention mechanism to overcome the small receptive field limitation of traditional convolutional methods.

## Limitations

- Traditional deep learning methods have small receptive fields and cannot capture long-range correlations.
- The standard Global Context (GC) block uses only one global context feature, which is insufficient for describing non-homogeneous structures.
