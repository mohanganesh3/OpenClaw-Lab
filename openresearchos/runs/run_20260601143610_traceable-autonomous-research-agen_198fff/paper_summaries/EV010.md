# EV010: SWE-agent: Agent-Computer Interfaces Enable Automated Software Engineering

Year: 2024

URL: https://arxiv.org/abs/2405.15793

## Core Claim

Demonstrates how agents can operate computer interfaces to solve software engineering tasks, with relevance to local experiment execution and reproducible command logs.

## Method Or Focus

- Agent-computer interfaces
- Automated software engineering
- Local experiment execution
- Reproducible command logs

## Limitations To Track

- Focus on local experiment execution may limit scalability
- Challenges in ensuring consistency of reproducible command logs

## Possible Gap

How agents can effectively interact with software development environments to perform complex software engineering tasks.
